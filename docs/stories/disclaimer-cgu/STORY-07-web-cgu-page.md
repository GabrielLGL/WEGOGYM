# STORY-07 — Page /cgu sur kore-app.net

## Description
Créer une page `/cgu` sur le site Next.js kore-app.net avec le texte complet des CGU, responsive et lisible sur mobile (sera affichée dans le WebView de l'app).

## Tâches techniques
1. Identifier la structure du site web (app router vs pages router)
2. Créer la page CGU avec les 8 sections
3. Style sobre, responsive, lisible en WebView mobile
4. Sections : Objet, Acceptation, Avertissement santé, Responsabilité, Données, Propriété intellectuelle, Modification, Contact

## Critères d'acceptation
- [ ] Page accessible à kore-app.net/cgu
- [ ] 8 sections complètes
- [ ] Responsive et lisible sur mobile
- [ ] Style cohérent avec le site existant

## Estimation
S (< 1h)
