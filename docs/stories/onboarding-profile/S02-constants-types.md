# S02 — Constantes et types (UserLevel, UserGoal)

## Description
Ajouter les types TypeScript et labels français pour les niveaux, objectifs et équipements dans `constants.ts`.

## Tâches techniques
- [ ] `constants.ts` : types `UserLevel`, `UserGoal`, `EquipmentType`
- [ ] `constants.ts` : labels français `USER_LEVEL_LABELS`, `USER_GOAL_LABELS`
- [ ] `constants.ts` : descriptions pour l'onboarding (ex: "Je commence la muscu")

## Critères d'acceptation

- [ ] Labels français corrects
