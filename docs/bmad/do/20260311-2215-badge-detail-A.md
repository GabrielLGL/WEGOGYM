## Comportement

- Clic simple sur n'importe quel badge (déverrouillé ou non) → BottomSheet détail
- Appui long sur badge **déverrouillé** → ShareBottomSheet (comportement inchangé)
- Le BottomSheet détail affiche :
  - L'icône en grand (colorée si déverrouillé, grisée si verrouillé)
  - La description i18n du badge
  - Le texte de condition avec le threshold (ex: "50 séances complétées")
  - Le label "Obtenu" ou "À débloquer"
