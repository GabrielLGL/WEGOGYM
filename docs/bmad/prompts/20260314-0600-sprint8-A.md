# Rapport — Gratitude Post-Séance (#40) — Groupe A — 20260314-0600

## Objectif
Ajouter une micro-interaction "Qu'est-ce qui t'a rendu fier ?" dans `WorkoutSummarySheet` (la feuille récap après séance). L'utilisateur choisit un emoji de satisfaction (💪 🔥 😌 😤) et optionnellement ajoute une note courte. Éphémère — pas de DB, état local seulement.

## Fichiers concernés
- `mobile/src/components/WorkoutSummarySheet.tsx` — MODIFIER : ajouter section gratitude en haut
- `mobile/src/i18n/fr.ts` — MODIFIER : ajouter section `gratitude.*`
- `mobile/src/i18n/en.ts` — MODIFIER : même structure

## Contexte technique

### Stack & Architecture
- React Native + Expo 52 + TypeScript strict (Fabric / New Architecture)
- `useColors()` / `useLanguage()` / `useHaptics()` hooks
- **Ne pas modifier les props existantes de WorkoutSummarySheet**
- `TextInput` natif pour la note courte
- Lire le fichier avant de coder pour comprendre la structure actuelle

### Structure existante de WorkoutSummarySheet
WorkoutSummarySheet est une BottomSheet qui s'ouvre après la fin d'une séance. Elle affiche des stats animées (XP, level, streak, badges). La gratitude s'ajoute **en haut**, avant les stats de séance.

## Implémentation

### État local (dans WorkoutSummarySheet)
```ts
const [selectedEmoji, setSelectedEmoji] = useState<string | null>(null)
const [gratitudeNote, setGratitudeNote] = useState('')
const [gratitudeSubmitted, setGratitudeSubmitted] = useState(false)
```

**Reset à chaque ouverture du sheet :**
```ts
useEffect(() => {
  if (!visible) {
    setSelectedEmoji(null)
    setGratitudeNote('')
    setGratitudeSubmitted(false)
  }
}, [visible])
```

### UI à ajouter EN HAUT du contenu de la BottomSheet

**Avant la soumission (`!gratitudeSubmitted`) :**
```tsx
{/* ── Section Gratitude ── */}
{!gratitudeSubmitted && (
  <View style={styles.gratitudeSection}>
    <Text style={styles.gratitudeQuestion}>{t.gratitude.question}</Text>
    <View style={styles.emojiRow}>
      {['💪', '🔥', '😌', '😤'].map(emoji => (
        <TouchableOpacity
          key={emoji}
          style={[
            styles.emojiBtn,
            selectedEmoji === emoji && styles.emojiBtnActive,
          ]}
          onPress={() => {
            haptics.onSelect()
            setSelectedEmoji(prev => prev === emoji ? null : emoji)
          }}
        >
          <Text style={styles.emojiText}>{emoji}</Text>
        </TouchableOpacity>
      ))}
    </View>
    {selectedEmoji && (
      <>
        <TextInput
          style={styles.gratitudeInput}
          placeholder={t.gratitude.placeholder}
          placeholderTextColor={colors.textMuted}
          value={gratitudeNote}
          onChangeText={setGratitudeNote}
          maxLength={120}
          multiline
        />
        <TouchableOpacity
          style={styles.gratitudeSubmitBtn}
          onPress={() => {
            haptics.onSuccess()
            setGratitudeSubmitted(true)
          }}
        >
          <Text style={styles.gratitudeSubmitText}>{t.gratitude.save}</Text>
        </TouchableOpacity>
      </>
    )}
  </View>
)}

{/* Après soumission */}
{gratitudeSubmitted && (
  <View style={styles.gratitudeDoneRow}>
    <Text style={styles.gratitudeDoneText}>{selectedEmoji} {t.gratitude.saved}</Text>
  </View>
)}

<View style={styles.sectionDivider} />
```

- La section gratitude s'affiche en haut du WorkoutSummarySheet
- Sélection d'emoji → TextInput apparaît

- Reset à chaque réouverture du sheet
