## Étapes

### 1. Créer `ShareBottomSheet.tsx`

Un BottomSheet réutilisable qui propose le choix texte / image :

```tsx
interface ShareBottomSheetProps {
  visible: boolean
  onClose: () => void
  onShareText: () => void
  onShareImage: () => void
}
```

Layout :
```
┌─────────────────────────────┐
│  Partager                   │
│─────────────────────────────│
│                             │
│  📝 Partager en texte       │  ← TouchableOpacity
│  Envoie un résumé texte     │
│                             │
│  🖼️ Partager en image       │  ← TouchableOpacity
│  Génère une belle carte     │
│                             │
└─────────────────────────────┘
```

### 2. Intégrer dans WorkoutSummarySheet.tsx

**Ajouter :**

**Position du bouton :** entre la section gamification et le bouton "Fermer" existant.

### 5. ViewShot off-screen pattern

Pour le rendu off-screen de la carte (pas visible à l'utilisateur mais capturable) :

```tsx
<View style={{ position: 'absolute', left: -9999 }}>
  <ViewShot ref={viewShotRef} options={{ format: 'png', quality: 1 }}>
    <ShareCard variant="workout" {...shareData} />
  </ViewShot>
</View>
```

Ceci permet de capturer l'image sans que la carte soit visible dans l'UI principale.

### 6. Gestion d'erreurs

- Try/catch autour de `shareText` et `shareImage`
- En cas d'erreur de capture : fallback sur le partage texte avec message `__DEV__` log
- Pas d'AlertDialog d'erreur (le share dialog natif gère les cas d'annulation)
