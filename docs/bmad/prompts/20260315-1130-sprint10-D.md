# Rapport — Timeline des Records Personnels (#99) — Groupe D — 20260315-1130

## Objectif
Créer un écran `StatsPRTimelineScreen` affichant une vue chronologique de tous les records personnels (PRs) avec une visualisation en timeline verticale. Chaque PR montre l'exercice, le poids, la date, et le gain par rapport au PR précédent. Accessible depuis StatsScreen.

## Implémentation

### prTimelineHelpers.ts

```ts
export interface PRTimelineEntry {
  setId: string
  exerciseId: string
  exerciseName: string
  weight: number
  reps: number
  date: number              // timestamp
  previousPR: number | null // poids du PR précédent pour cet exercice
  gain: number | null       // weight - previousPR (kg)
  gainPercent: number | null // ((weight - previousPR) / previousPR) × 100
}

export interface PRTimelineMonth {
  month: string             // "Mars 2026"
  year: number
  monthIndex: number        // 0-11
  entries: PRTimelineEntry[]
  totalPRs: number
}

/**
 * Extrait tous les PRs et les organise en timeline mensuelle.
 */
export function buildPRTimeline(
  sets: Array<{ id: string; weight: number; reps: number; isPr: boolean; exerciseId: string; createdAt: Date | number }>,
  exercises: Array<{ id: string; name: string }>,
): PRTimelineMonth[]
```

**Algorithme :**
1. Filtrer les sets où `isPr === true`
2. Trier par date décroissante (plus récent en premier)
3. Pour chaque PR, trouver le PR précédent du même exercice (le set avec isPr=true le plus récent avant cette date pour le même exerciseId)
4. Calculer `gain` et `gainPercent`
5. Trouver le nom de l'exercice via `exercises`
6. Grouper par mois (année + mois)
7. Retourner trié par date décroissante

### withObservables pattern
```ts
const enhance = withObservables([], () => ({
  sets: database.get<WorkoutSet>('sets').query(
    Q.where('is_pr', true),
    Q.on('histories', Q.and(
      Q.where('deleted_at', null),
      Q.or(Q.where('is_abandoned', null), Q.where('is_abandoned', false)),
    )),
    Q.sortBy('created_at', Q.desc),
  ).observe(),
  exercises: database.get<Exercise>('exercises').query().observe(),
}))
```
