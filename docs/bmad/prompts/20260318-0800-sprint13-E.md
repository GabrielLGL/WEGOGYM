# Rapport — Objectifs Hebdo (#115) — Groupe E — 20260318-0800

## Objectif
Ajouter une carte "Objectifs de la semaine" sur `HomeScreen` avec des objectifs configurables : nombre de seances cible par semaine et volume cible hebdomadaire. Affiche la progression avec des barres de progression.

## Fichiers concernes
- `mobile/src/model/utils/weeklyGoalsHelpers.ts` — NOUVEAU
- `mobile/src/screens/HomeScreen.tsx` — MODIFIER : ajouter carte objectifs
- `mobile/src/i18n/fr.ts` — MODIFIER : ajouter cles `home.weeklyGoals.*`
- `mobile/src/i18n/en.ts` — MODIFIER : meme structure

## Contexte technique
- React Native + Expo 52 + TypeScript strict
- `useColors()` / `useLanguage()` hooks
- HomeScreen a deja `histories` et `sets` dans ses props
- Les objectifs sont hardcodes pour l'instant (pas de settings) :
  - Seances cible : 4/semaine
  - Volume cible : 20000 kg/semaine
- La carte va **apres** la carte readiness (Sprint 13-A), ou apres la carte fatigue/heatmap existante
- Ne pas modifier withObservables

## Implementation

### weeklyGoalsHelpers.ts

```ts
export interface WeeklyGoalProgress {
  sessions: {
    current: number
    target: number
    percent: number       // 0-100, cap a 100
  }
  volume: {
    current: number
    target: number
    percent: number
  }
  daysRemaining: number   // jours restants dans la semaine (lun-dim)
  weekLabel: string       // ex: '11-17 mars'
  overallPercent: number  // moyenne des deux
}

/**
 * Calcule la progression des objectifs hebdomadaires.
 *
 * @param histories — seances non-abandonnees
 * @param sets — tous les sets
 * @param sessionsTarget — objectif seances (defaut 4)
 * @param volumeTarget — objectif volume en kg (defaut 20000)
 *
 * Algorithme :
 * 1. Determiner la semaine en cours (lundi a dimanche)
 * 2. Filtrer histories et sets de la semaine en cours
 * 3. sessions.current = nombre de histories cette semaine
 * 4. volume.current = somme(weight * reps) des sets cette semaine
 * 5. Calculer les pourcentages (cap a 100)
 * 6. daysRemaining = dimanche - aujourd'hui
 * 7. weekLabel = "DD-DD mois" format
 */
export function computeWeeklyGoals(
  histories: Array<{ startedAt: Date | number; isAbandoned: boolean }>,
  sets: Array<{ weight: number; reps: number; createdAt: Date | number }>,
  sessionsTarget?: number,
  volumeTarget?: number,
): WeeklyGoalProgress

/**
 * Retourne le lundi de la semaine contenant la date donnee.
 */
export function getWeekStart(date: Date): Date
```

### Section UI dans HomeScreen

```tsx
{weeklyGoals && (
  <View style={styles.goalsCard}>
    <View style={styles.goalsHeader}>
      <Text style={styles.goalsTitle}>{t.home.weeklyGoals.title}</Text>
      <Text style={styles.goalsWeek}>{weeklyGoals.weekLabel}</Text>
    </View>

    {/* Sessions goal */}
    <View style={styles.goalRow}>
      <Text style={styles.goalLabel}>{t.home.weeklyGoals.sessions}</Text>
      <View style={styles.goalBarTrack}>
        <View style={[styles.goalBarFill, {
          width: `${weeklyGoals.sessions.percent}%`,
          backgroundColor: weeklyGoals.sessions.percent >= 100 ? '#10B981' : colors.primary,
        }]} />
      </View>
      <Text style={styles.goalCount}>
        {weeklyGoals.sessions.current}/{weeklyGoals.sessions.target}
      </Text>
    </View>

    {/* Volume goal */}
    <View style={styles.goalRow}>
      <Text style={styles.goalLabel}>{t.home.weeklyGoals.volume}</Text>
      <View style={styles.goalBarTrack}>
        <View style={[styles.goalBarFill, {
          width: `${weeklyGoals.volume.percent}%`,
          backgroundColor: weeklyGoals.volume.percent >= 100 ? '#10B981' : colors.primary,
        }]} />
      </View>
      <Text style={styles.goalCount}>
        {(weeklyGoals.volume.current / 1000).toFixed(1)}t / {(weeklyGoals.volume.target / 1000).toFixed(0)}t
      </Text>
    </View>

    <Text style={styles.goalsDaysLeft}>
      {weeklyGoals.daysRemaining} {t.home.weeklyGoals.daysLeft}
    </Text>
  </View>
)}
```
