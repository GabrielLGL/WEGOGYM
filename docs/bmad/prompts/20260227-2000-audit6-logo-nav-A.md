# Rapport — Audit 6 Site Kore — Groupe A : Logo aria-hidden + Nav aria-label — 20260227-2000

## Objectif
1. Le composant `KoreLogo` expose `role="img" aria-label="Logo Kore"` mais est **toujours affiché à côté du texte visible "KORE"** → doublon annoncé aux lecteurs d'écran. Il doit être décoratif (`aria-hidden="true"`).
2. La `<nav>` sticky de `HeroSection` n'a pas d'`aria-label` → les lecteurs d'écran ne distinguent pas cette nav des autres landmarks. De plus, le lien logo pointe vers `href="#"` au lieu de `href="/"`.

## Fichiers concernés
- `web/src/components/KoreLogo.tsx`
- `web/src/components/sections/HeroSection.tsx`

## Contexte technique
- `KoreLogo` est utilisé dans : `HeroSection.tsx`, `FooterSection.tsx`, `privacy/page.tsx`, `not-found.tsx` — dans **tous** les cas, il est accompagné du texte visible "KORE", donc toujours décoratif.
- Le site utilise Next.js App Router avec Tailwind CSS v4.
- `npx tsc --noEmit` (depuis `web/`) doit rester à 0 erreur.
- Ne pas modifier les styles, animations, ou structure JSX — uniquement les attributs aria et le href.

## Étapes

### 1. `web/src/components/KoreLogo.tsx` — lignes 15–16

Supprimer `role="img"` et `aria-label="Logo Kore"`, remplacer par `aria-hidden="true"` :

```tsx
// AVANT
<svg
  className={`animate-[pulseLogo_3s_infinite_ease-in-out] ${className}`}
  width={size}
  height={size}
  viewBox="0 0 100 100"
  xmlns="http://www.w3.org/2000/svg"
  role="img"
  aria-label="Logo Kore"
>

// APRÈS
<svg
  className={`animate-[pulseLogo_3s_infinite_ease-in-out] ${className}`}
  width={size}
  height={size}
  viewBox="0 0 100 100"
  xmlns="http://www.w3.org/2000/svg"
  aria-hidden="true"
>
```

### 2. `web/src/components/sections/HeroSection.tsx` — ligne 51

Ajouter `aria-label="Navigation principale"` sur la `<nav>` :

```tsx
// AVANT
<nav
  className={`fixed top-0 left-0 right-0 z-50 py-3 transition-all duration-300 ${...}`}
>

// APRÈS
<nav
  aria-label="Navigation principale"
  className={`fixed top-0 left-0 right-0 z-50 py-3 transition-all duration-300 ${...}`}
>
```

### 3. `web/src/components/sections/HeroSection.tsx` — ligne 59

Corriger `href="#"` → `href="/"` sur le lien logo de la nav :

```tsx
// AVANT
<a href="#" className="flex items-center gap-2 no-underline">

// APRÈS
<a href="/" className="flex items-center gap-2 no-underline">
```

## Contraintes
- Ne modifier QUE les attributs indiqués (aria-hidden, aria-label, href)
- Ne pas toucher aux styles, animations pulseLogo, gradients SVG
- Ne pas modifier les autres occurrences de KoreLogo dans d'autres fichiers (le changement dans KoreLogo.tsx les couvre automatiquement)

## Critères de validation
- `npx tsc --noEmit` (depuis `web/`) → 0 erreur
- Lecteur d'écran : le logo n'est plus annoncé (aria-hidden), seul le texte "KORE" est lu
- La `<nav>` est annoncée comme "Navigation principale" par le lecteur d'écran
- Le lien logo de la nav redirige vers `/` au lieu de `#`
