## Implémentation

### plateauHelpers.ts

```ts
export type PlateauStrategySuggestion =
  | 'deload'           // Faire une semaine de décharge
  | 'vary_reps'        // Changer la plage de reps (ex: 3×10 → 5×5)
  | 'add_volume'       // Ajouter des séries d'échauffement
  | 'rest_pause'       // Technique pause-repos
  | 'progressive'      // Micro-chargement (0.5kg à la fois)

export interface PlateauData {
  isPlateauing: boolean
  sessionsSinceLastPR: number      // séances depuis le dernier PR
  daysSinceLastProgress: number    // jours depuis la dernière amélioration du 1RM
  currentBest1RM: number
  strategies: PlateauStrategySuggestion[]  // 1-2 suggestions
}

/**
 * Détecte si l'utilisateur est en plateau sur cet exercice.
 * Retourne null si moins de 5 sets au total.
 */
export function computePlateauAnalysis(sets: WorkoutSet[]): PlateauData | null
```

**Algorithme :**
1. Si < 5 sets → retourner `null`
2. Trier sets par `createdAt` décroissant
3. Calculer 1RM Epley pour chaque set
4. `currentBest1RM` = max 1RM de tous les sets
5. `lastPRDate` = date du set avec le meilleur 1RM
6. `daysSinceLastProgress` = `Math.floor((Date.now() - lastPRDate.getTime()) / DAY_MS)`
7. Compter `sessionsSinceLastPR` : nombre de dates uniques de sets après `lastPRDate`
8. **Plateau détecté si** : `sessionsSinceLastPR >= 3` ET `daysSinceLastProgress >= 21`
9. Suggestions selon le contexte :
   - `sessionsSinceLastPR >= 6` → `['deload', 'vary_reps']`
   - `sessionsSinceLastPR >= 3` → `['progressive', 'vary_reps']`
10. `isPlateauing = sessionsSinceLastPR >= 3 && daysSinceLastProgress >= 21`

**Import :**
```ts

```
