# Rapport — Défis Personnels / Boss Fights (#5) — Groupe C — 20260314-0600

## Objectif
Créer un écran `PersonalChallengesScreen` affichant des défis personnels épiques calculés depuis les données existantes : "Atteins 100kg au squat", "Soulève 10 000 kg en une semaine", etc. Les défis sont générés dynamiquement selon les PRs actuels et complétés automatiquement. Accessible via un bouton dans la section "Outils" du HomeScreen.

## Fichiers concernés
- `mobile/src/model/utils/personalChallengesHelpers.ts` — NOUVEAU : définitions + calcul progression
- `mobile/src/screens/PersonalChallengesScreen.tsx` — NOUVEAU : écran principal

## Implémentation

### personalChallengesHelpers.ts

```ts
export interface PersonalChallenge {
  id: string
  icon: string              // Ionicons name
  title: string             // clé i18n
  description: string       // clé i18n
  progress: number          // 0-1 normalisé
  currentValue: number      // valeur actuelle
  targetValue: number       // valeur cible
  completed: boolean        // progress >= 1
  unit: string              // 'kg' | 'séances' | 'semaines' | 'PRs' | 'level'
  difficulty: 'easy' | 'medium' | 'hard' | 'legendary'
}

/**
 * Génère la liste des défis personnels dynamiques.
 * Les cibles s'adaptent au niveau de l'utilisateur.
 */
export function computePersonalChallenges(
  user: User,
  totalHistories: number,
): PersonalChallenge[]
```

**12 défis définis :**

| id | Métrique | Cible | Difficulté |
|----|----------|-------|------------|
| `first_50` | totalHistories | 50 | easy |
| `first_100` | totalHistories | 100 | medium |
| `first_250` | totalHistories | 250 | hard |
| `tonnage_50k` | user.totalTonnage | 50 000 kg | easy |
| `tonnage_250k` | user.totalTonnage | 250 000 kg | medium |
| `tonnage_1M` | user.totalTonnage | 1 000 000 kg | legendary |
| `prs_25` | user.totalPrs | 25 | easy |
| `prs_100` | user.totalPrs | 100 | medium |
| `prs_500` | user.totalPrs | 500 | hard |
| `streak_4` | user.bestStreak | 4 semaines | easy |
| `streak_26` | user.bestStreak | 26 semaines | hard |
| `level_50` | user.level | 50 | legendary |

**Tri** : défis non-complétés en premier (par progression décroissante), puis complétés.

### withObservables pattern
```ts
const enhance = withObservables([], () => ({
  user: observeCurrentUser(),
  histories: database.get<History>('histories').query(
    Q.where('deleted_at', null),
    Q.or(Q.where('is_abandoned', null), Q.where('is_abandoned', false)),
  ).observe(),
}))
```
