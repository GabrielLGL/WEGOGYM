# Rapport — SEO technique — Groupe A — 20260310-1430

## Objectif
Corriger toutes les lacunes techniques SEO du site kore-app.net pour permettre l'indexation par Google et améliorer le ranking.

## Fichiers concernés
- `web/src/app/sitemap.ts`
- `web/src/app/robots.ts`
- `web/src/app/layout.tsx` (metadata + structured data)
- `web/src/app/privacy/page.tsx` (ajouter canonical)
- `web/src/app/contact/layout.tsx` (ajouter canonical)
- `web/src/app/cgu/page.tsx` (ajouter canonical)
- `web/src/app/mentions-legales/page.tsx` (ajouter canonical)

## Contexte technique
- Next.js 15 App Router
- Site déployé sur Vercel à https://kore-app.net
- Metadata via export `metadata` dans layout/page
- Structured data via JSON-LD dans `<head>` du root layout

## Étapes

### 1. Compléter le sitemap (`sitemap.ts`)
Ajouter TOUTES les pages publiques :
```ts
export default function sitemap(): MetadataRoute.Sitemap {
  return [
    { url: siteUrl, lastModified: new Date("2026-03-10"), changeFrequency: "weekly", priority: 1 },
    { url: `${siteUrl}/contact`, lastModified: new Date("2026-03-10"), changeFrequency: "monthly", priority: 0.5 },
    { url: `${siteUrl}/privacy`, lastModified: new Date("2026-01-01"), changeFrequency: "yearly", priority: 0.3 },
    { url: `${siteUrl}/mentions-legales`, lastModified: new Date("2026-01-01"), changeFrequency: "yearly", priority: 0.3 },
    { url: `${siteUrl}/cgu`, lastModified: new Date("2026-01-01"), changeFrequency: "yearly", priority: 0.3 },
  ];
}
```

### 2. Optimiser les keywords dans le root metadata (`layout.tsx`)
Ajouter des mots-clés long-tail FR que les gens recherchent vraiment :
```ts
keywords: [
  "musculation",
  "application musculation gratuite",
  "appli musculation android",
  "appli musculation offline",
  "tracker musculation",
  "carnet entraînement musculation",
  "suivi séance musculation",
  "programme musculation personnalisé",
  "gym tracker gratuit",
  "workout tracker android",
  "coach musculation",
  "Kore musculation",
  "kore app",
],
```

### 3. Améliorer la meta description (`layout.tsx`)
Inclure les mots-clés cibles dans la description :
```
"Kore — Application de musculation gratuite et 100% offline pour Android. Crée tes programmes, suis tes performances et progresse séance après séance. Tracker de musculation rapide et intuitif."
```

### 4. Améliorer le title tag (`layout.tsx`)
```
"Kore — Application de musculation gratuite | Tracker offline Android"
```

### 5. Ajouter canonical sur chaque page interne
Dans chaque page qui exporte metadata, ajouter :
```ts
alternates: { canonical: "/privacy" }  // ou /contact, /cgu, /mentions-legales
```

### 6. Enrichir le structured data (`layout.tsx`)
Ajouter un second bloc JSON-LD de type `WebSite` avec `SearchAction` potentiel, et un `FAQPage` (quand le FAQ sera ajouté par le Groupe B).

Ajouter aussi :
```json
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "Kore",
  "url": "https://kore-app.net",
  "description": "Application de musculation gratuite et offline pour Android",
  "inLanguage": "fr-FR"
}
```

### 7. Mettre à jour OG title et Twitter title
Aligner avec le nouveau title optimisé SEO.
