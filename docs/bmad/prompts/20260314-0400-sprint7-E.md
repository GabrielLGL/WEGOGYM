# Rapport — Tips Contextuels (#30) — Groupe E — 20260314-0400

## Objectif

## Implémentation

### workoutTipsHelpers.ts

```ts
// Tips par groupe musculaire (en clés i18n)
// Chaque muscle a un tableau de tips identifiés par clé
export const MUSCLE_TIPS: Record<string, string[]> = {
  'Pecs':        ['tips.pecs.0', 'tips.pecs.1'],
  'Dos':         ['tips.dos.0', 'tips.dos.1'],
  'Quadriceps':  ['tips.quads.0', 'tips.quads.1'],
  'Ischios':     ['tips.ischios.0'],
  'Epaules':     ['tips.epaules.0', 'tips.epaules.1'],
  'Biceps':      ['tips.biceps.0'],
  'Triceps':     ['tips.triceps.0'],
  'Trapèzes':    ['tips.trapezes.0'],
  'Abdos':       ['tips.abdos.0'],
  'Mollets':     ['tips.mollets.0'],
  'Cardio':      ['tips.cardio.0'],
}

// Tip générique si aucun muscle mappé
export const GENERIC_TIPS = ['tips.generic.0', 'tips.generic.1', 'tips.generic.2']

/**
 * Retourne la clé i18n du tip pour un exercice donné.
 * Sélection déterministe basée sur l'exercice ID.
 */
export function getTipKeyForExercise(
  exerciseId: string,
  muscles: string[],
): string {
  const candidates: string[] = []
  for (const muscle of muscles) {
    const tips = MUSCLE_TIPS[muscle] ?? []
    candidates.push(...tips)
  }
  if (candidates.length === 0) {
    // Fallback générique, déterministe
    const hash = exerciseId.split('').reduce((a, c) => a + c.charCodeAt(0), 0)
    return GENERIC_TIPS[hash % GENERIC_TIPS.length]
  }
  const hash = exerciseId.split('').reduce((a, c) => a + c.charCodeAt(0), 0)
  return candidates[hash % candidates.length]
}
```

### Intégration dans WorkoutScreen

**Import :**
```ts
import { getTipKeyForExercise } from '../model/utils/workoutTipsHelpers'
```

**Stratégie d'affichage :**
Lire WorkoutScreen.tsx pour comprendre la structure. Généralement, les exercices sont rendus dans une FlatList ou ScrollView avec un composant par exercice. Ajouter le tip dans chaque item d'exercice, juste sous le nom, sous la forme d'une petite ligne de texte avec une icône `💡`.

Si WorkoutScreen rend les exercices avec un mapping, ajouter dans chaque item :
```tsx
{/* ── Tip contextuel ── */}
{exercise?.muscles && (
  <View style={styles.tipRow}>
    <Text style={styles.tipIcon}>💡</Text>
    <Text style={styles.tipText} numberOfLines={2}>
      {t.workoutTips[getTipKeyForExercise(exercise.id, exercise.muscles)
        .replace('tips.', '').replace('.', '_')] ?? ''}
    </Text>
  </View>
)}
```

**Alternative plus simple** si la structure est complexe : utiliser un `flattenObject` ou simplement utiliser un getter :
```ts
function getTipText(key: string, t: typeof translations.fr): string {
  const parts = key.split('.')  // ['tips', 'pecs', '0']
  // naviguer dans t.workoutTips selon la structure choisie
  return (t.workoutTips as Record<string, string>)[parts.slice(1).join('_')] ?? ''
}
```
