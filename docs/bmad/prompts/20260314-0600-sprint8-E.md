## Implémentation

### Logique de calcul des percentiles (dans useMemo)

Les percentiles sont estimés depuis la valeur normalisée de chaque axe `axis.value` (0-1) :

```ts
function estimatePercentile(normalizedValue: number): number {
  // Estimation statistique basée sur une distribution normale
  // La plupart des gens sont dans la moyenne (0.3-0.5)
  if (normalizedValue >= 0.9) return 95
  if (normalizedValue >= 0.75) return 80
  if (normalizedValue >= 0.6) return 65
  if (normalizedValue >= 0.45) return 50
  if (normalizedValue >= 0.3) return 35
  if (normalizedValue >= 0.15) return 20
  return 10
}

function percentileLabel(pct: number): 'top5' | 'top10' | 'top20' | 'top35' | 'top50' | 'bottom50' {
  if (pct >= 95) return 'top5'
  if (pct >= 80) return 'top10' // top 20% mais label top 10 pour la rondeur
  if (pct >= 65) return 'top20' // estimé top 35% mais afficher top 35
  if (pct >= 50) return 'top35'
  if (pct >= 35) return 'top50'
  return 'bottom50'
}
```

**Simplification** : utiliser directement une map `value → label` :
```ts
function getPercentileLabel(value: number): string {
  if (value >= 0.9) return t.hexagon.percentiles.top5
  if (value >= 0.75) return t.hexagon.percentiles.top20
  if (value >= 0.5) return t.hexagon.percentiles.top35
  if (value >= 0.3) return t.hexagon.percentiles.top50
  return t.hexagon.percentiles.bottom50
}
```

### Section UI à ajouter dans StatsHexagonScreen

Ajouter **après** la section "Détail par axe" existante :

```tsx
{/* ── Section Percentiles ── */}
<View style={styles.percentileSection}>
  <Text style={styles.sectionTitle}>{t.hexagon.percentiles.sectionTitle}</Text>
  <Text style={styles.percentileSubtitle}>{t.hexagon.percentiles.subtitle}</Text>

  {axes.map(axis => (
    <View key={axis.label} style={styles.percentileRow}>
      <View style={styles.percentileLeft}>
        <Ionicons name={axis.icon as any} size={16} color={axis.color} />
        <Text style={styles.percentileAxisLabel}>{axis.label}</Text>
      </View>
      <View style={[
        styles.percentileBadge,
        { backgroundColor: axis.value >= 0.75 ? colors.primary + '22' : colors.cardSecondary }
      ]}>
        <Text style={[
          styles.percentileBadgeText,
          { color: axis.value >= 0.75 ? colors.primary : colors.textSecondary }
        ]}>
          {getPercentileLabel(axis.value)}
        </Text>
      </View>
    </View>
  ))}

  <Text style={styles.percentileDisclaimer}>{t.hexagon.percentiles.disclaimer}</Text>
</View>
```
