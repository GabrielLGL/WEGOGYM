## Objectif
Convertir les inline styles de `MilestoneCelebration.tsx` en `StyleSheet.create()`,
conformément au pattern standard du projet (cf CLAUDE.md §5).

## Fichiers concernés
- `mobile/src/components/MilestoneCelebration.tsx`

## Contexte technique
Le composant utilise actuellement des objets de style inline directement dans le JSX
(ex: `style={{ alignItems: 'center', paddingVertical: spacing.md }}`).

Le pattern standard du projet est :
```tsx
function useStyles(colors: ThemeColors) {
  return StyleSheet.create({
    container: { ... },
  })
}
```
appelé dans le composant : `const styles = useStyles(colors)`

Le composant a déjà `useColors()` et importe `spacing`, `fontSize` depuis `'../theme'`.
Il faut ajouter `StyleSheet` depuis `'react-native'` et créer la fonction `useStyles`.

## Étapes
1. Lire `mobile/src/components/MilestoneCelebration.tsx` (fichier court ~53 lignes)
2. Identifier tous les blocs `style={{ ... }}` inline dans le JSX
3. Créer `function useStyles(colors: ThemeColors)` avec `StyleSheet.create({ ... })`
4. Remplacer chaque style inline par `styles.nomDuStyle`
5. Ajouter `StyleSheet` à l'import `react-native` et `ThemeColors` à l'import `'../theme'`
6. Vérifier : `npx tsc --noEmit` → zéro erreur
