## Implementation

### workoutReadinessHelpers.ts

```ts
export interface ReadinessResult {
  score: number           // 0-100
  level: 'optimal' | 'good' | 'moderate' | 'low'
  components: {
    recovery: number      // 0-100 (moyenne recuperation musculaire)
    fatigue: number       // 0-100 (inverse de l'ACWR — plus c'est haut, mieux c'est)
    consistency: number   // 0-100 (basee sur le streak et la regularite)
  }
  recommendation: string  // cle i18n
}

/**
 * Calcule le score de readiness composite.
 *
 * @param sets — tous les sets
 * @param exercises — tous les exercices
 * @param histories — toutes les seances (non-abandonnees)
 *
 * Algorithme :
 * 1. Recovery (40% du score) :
 *    - Calculer la moyenne des recoveryPercent de computeMuscleRecovery
 *    - Si aucun muscle entraine recemment → 100
 * 2. Fatigue (35% du score) :
 *    - Calculer l'ACWR via computeFatigueIndex
 *    - ACWR < 0.8 → 90, ACWR 0.8-1.3 → 70, ACWR 1.3-1.5 → 40, ACWR > 1.5 → 15
 * 3. Consistency (25% du score) :
 *    - Compter les jours d'entrainement sur les 14 derniers jours
 *    - 0j → 20, 1-2j → 40, 3-4j → 70, 5-6j → 90, 7+j → 80 (surentrainement)
 * 4. Score = recovery*0.40 + fatigue*0.35 + consistency*0.25
 * 5. Level : >=80 → 'optimal', >=60 → 'good', >=40 → 'moderate', <40 → 'low'
 * 6. Recommendation selon level :
 *    - optimal → 'readiness.rec.goHard'
 *    - good → 'readiness.rec.normalSession'
 *    - moderate → 'readiness.rec.lightSession'
 *    - low → 'readiness.rec.restDay'
 */
export function computeReadiness(
  sets: Array<{ weight: number; reps: number; exerciseId: string; createdAt: Date | number }>,
  exercises: Array<{ id: string; muscles: string[] }>,
  histories: Array<{ startedAt: Date | number; isAbandoned: boolean }>,
): ReadinessResult
```

### Section UI dans HomeScreen

Carte circulaire avec le score au centre, 3 sous-indicateurs en dessous, et la recommandation.

```tsx
{readinessData && (
  <View style={styles.readinessCard}>
    <View style={styles.readinessHeader}>
      <Text style={styles.readinessTitle}>{t.home.readiness.title}</Text>
      <Text style={styles.readinessLevel}>
        {t.home.readiness.levels[readinessData.level]}
      </Text>
    </View>
    <View style={styles.readinessScoreContainer}>
      <Text style={[styles.readinessScore, { color: getReadinessColor(readinessData.level, colors) }]}>
        {readinessData.score}
      </Text>
      <Text style={styles.readinessMax}>/100</Text>
    </View>
    <View style={styles.readinessComponents}>
      <ReadinessBar label={t.home.readiness.recovery} value={readinessData.components.recovery} colors={colors} />
      <ReadinessBar label={t.home.readiness.fatigue} value={readinessData.components.fatigue} colors={colors} />
      <ReadinessBar label={t.home.readiness.consistency} value={readinessData.components.consistency} colors={colors} />
    </View>
    <Text style={styles.readinessRec}>{t.home.readiness.recommendations[readinessData.level]}</Text>
  </View>
)}
```

**ReadinessBar** : mini-barre horizontale avec label et valeur.

**Couleurs :**
- optimal → colors.primary
- good → '#10B981'
- moderate → '#F59E0B'
- low → colors.danger

**Styles :** carte classique (backgroundColor: colors.card, borderRadius: borderRadius.lg, padding: spacing.md). Score en fontSize.jumbo au centre.
