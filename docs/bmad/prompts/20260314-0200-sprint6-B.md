# Rapport — Bulletin Mensuel (#87) — Groupe B — 20260314-0200

## Objectif
Créer un écran `MonthlyBulletinScreen` qui affiche un bilan mensuel gamifié avec des notes par catégorie (Régularité A, Force B+, Volume C, Équilibre B). Accessible depuis StatsScreen. Le bilan compare le mois en cours aux mois précédents et attribue automatiquement des lettres A/B/C/D selon les percentiles.

## Implémentation

### monthlyBulletinHelpers.ts

```ts
export type Grade = 'A' | 'B' | 'C' | 'D'
export type GradePlus = 'A+' | 'A' | 'B+' | 'B' | 'C+' | 'C' | 'D'

export interface MonthlyGrade {
  category: 'regularite' | 'force' | 'volume' | 'equilibre'
  grade: GradePlus
  value: number        // valeur brute ce mois
  avg: number          // moyenne des mois précédents
  pctVsAvg: number     // % vs moyenne (peut être négatif)
}

export interface MonthlyBulletin {
  month: string        // ex: "Mars 2026"
  monthIndex: number   // 0-11
  year: number
  grades: MonthlyGrade[]
  overallGrade: GradePlus
  comment: string      // commentaire auto généré
}

/**
 * Calcule le bulletin du mois en cours.
 * Retourne null si moins de 2 mois de données.
 */
export function computeMonthlyBulletin(
  histories: History[],
  sets: WorkoutSet[],
  exercises: Exercise[],
  language: 'fr' | 'en',
): MonthlyBulletin | null
```

**Algorithme :**
1. Définir le mois courant (année + mois)
2. Grouper les histories par mois
3. Si < 2 mois → retourner `null`
4. Pour le mois courant, calculer :
   - `regularite` : nombre de séances ce mois
   - `force` : nombre de PRs (sets.isPr) ce mois
   - `volume` : tonnage total (Σ weight*reps) ce mois
   - `equilibre` : nombre de groupes musculaires distincts ce mois
5. Calculer la moyenne sur les mois précédents pour chaque métrique
6. Attribuer une note selon le percentile vs la moyenne :
   - ≥ 130% → A+, ≥ 110% → A, ≥ 90% → B+, ≥ 70% → B, ≥ 50% → C+, ≥ 30% → C, < 30% → D
7. `overallGrade` = note majoritaire (ou moyenne arrondie)
8. Générer un `comment` selon l'overall grade (phrase d'encouragement)

### withObservables pattern
```ts
const enhance = withObservables([], () => ({
  sets: database.get<WorkoutSet>('sets').query(
    Q.on('histories', Q.and(
      Q.where('deleted_at', null),
      Q.or(Q.where('is_abandoned', null), Q.where('is_abandoned', false)),
    ))
  ).observe(),
  exercises: database.get<Exercise>('exercises').query().observe(),
  histories: database.get<History>('histories').query(
    Q.where('deleted_at', null),
    Q.or(Q.where('is_abandoned', null), Q.where('is_abandoned', false)),
  ).observe(),
}))
```
