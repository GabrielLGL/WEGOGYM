## Implémentation

### overtrainingHelpers.ts

```ts
export type OvertrainingSignal =
  | 'no_rest_7days'        // 7 séances ou plus sans jour de repos de 2 jours consécutifs
  | 'volume_spike_3weeks'  // volume moyen 3 dernières semaines > 150% des 3 semaines précédentes
  | 'high_frequency'       // > 7 séances sur les 7 derniers jours
  | null

export interface OvertrainingData {
  signal: OvertrainingSignal
  detail: number    // la valeur numérique pertinente (jours consécutifs, ratio %, séances/semaine)
}

/**
 * Détecte les signaux de surentraînement.
 * Retourne null si données insuffisantes (< 10 séances).
 */
export function computeOvertrainingAlert(
  histories: History[],
  sets: WorkoutSet[],
): OvertrainingData | null
```

**Algorithme :**
1. Si `histories.length < 10` → retourner `null`
2. Trier histories par `startTime` décroissant
3. **Signal `high_frequency`** : compter séances dans les 7 derniers jours. Si > 7 → signal.
4. **Signal `no_rest_7days`** : vérifier les 14 derniers jours. Si aucun écart de ≥ 2 jours consécutifs entre 2 séances → signal. `detail` = nombre de jours sans repos.
5. **Signal `volume_spike_3weeks`** : calculer le tonnage (Σweight*reps) sur les 3 dernières semaines vs les 3 semaines précédentes. Si ratio > 1.5 → signal. `detail` = ratio * 100.
6. Priorité : `high_frequency` > `no_rest_7days` > `volume_spike_3weeks`
7. Si aucun signal → retourner `null`

### Intégration dans HomeScreen

**Import :**
```ts
import { computeOvertrainingAlert } from '../model/utils/overtrainingHelpers'
```

**useMemo** (après `motivationData`) :
```ts
const overtrainingAlert = useMemo(
  () => computeOvertrainingAlert(histories, sets),
  [histories, sets],
)
```

**Placement :** Ajouter **après** la carte motivation et **avant** `{/* ── Weekly Report Card ── */}` :
```tsx
{/* ── Alerte surentraînement ── */}
{overtrainingAlert && (
  <View style={styles.overtrainingCard}>
    <View style={styles.overtrainingHeader}>
      <Ionicons name="warning-outline" size={20} color={colors.warning} />
      <Text style={styles.overtrainingTitle}>{t.overtraining.title}</Text>
    </View>
    <Text style={styles.overtrainingText}>
      {overtrainingAlert.signal === 'high_frequency' && t.overtraining.highFrequency.replace('{n}', String(overtrainingAlert.detail))}
      {overtrainingAlert.signal === 'no_rest_7days' && t.overtraining.noRest}
      {overtrainingAlert.signal === 'volume_spike_3weeks' && t.overtraining.volumeSpike.replace('{n}', String(Math.round(overtrainingAlert.detail)))}
    </Text>
    <Text style={styles.overtrainingAdvice}>{t.overtraining.advice}</Text>
  </View>
)}
```
