## Consignes pour chaque fichier test

### 1. exerciseFrequencyHelpers.test.ts
Lire `exerciseFrequencyHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne vide si aucun set
- Compte correct par exercice
- Tri par frequence decroissante
- Exercices negliges detectes (>30j)
- Filtre par periode (30j, 90j, tout)
- Trend correct (increasing/decreasing/stable)

### 2. weeklyGoalsHelpers.test.ts
Lire `weeklyGoalsHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne 0/0 si aucune donnee cette semaine
- Sessions comptees correctement
- Volume calcule (weight * reps)
- Pourcentages caps a 100
- getWeekStart retourne bien un lundi
- daysRemaining correct

### 3. workoutReadinessHelpers.test.ts
Lire `workoutReadinessHelpers.ts`, identifier les fonctions exportees.
Tests :
- Score 100 si aucune seance recente (full recovery)
- Level 'optimal' pour score >= 80
- Level 'low' pour score < 40
- Composants recovery/fatigue/consistency dans [0, 100]
- Score arrondi a l'entier

### 4. muscleBalanceHelpers.test.ts
Lire `muscleBalanceHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne vide si aucun set
- 4 paires musculaires generees
- Ratio 1.0 quand volumes egaux → 'balanced'
- Ratio > 1.5 → 'imbalanced'
- Filtre par periode
- overallBalance dans [0, 100]

### 5. trainingDensityHelpers.test.ts
Lire `trainingDensityHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne 0 si aucun set
- Densite = totalVolume / elapsedMinutes
- elapsedMinutes minimum 1
- Comparison 'faster'/'slower'/'similar' correcte
- formatDensity retourne "X kg/min"

### 6. exerciseAlternativesHelpers.test.ts
Lire `exerciseAlternativesHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne vide si exercice inconnu
- Alternatives partagent des muscles communs
- L'exercice source n'est pas dans ses propres alternatives
- Maximum d'alternatives raisonnable

### 7. durationPredictorHelpers.test.ts
Lire `durationPredictorHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne null si pas d'historique
- Prediction basee sur durees precedentes de la meme session
- Gere une seule seance passee
- Prediction raisonnable (pas negative, pas > 300min)
