# Rapport — Streak Heatmap (#108) — Groupe C — 20260317-0800

## Objectif
Ajouter une carte "Streak heatmap" sur `HomeScreen` affichant une grille de type GitHub (contribution grid) montrant les jours d'entrainement sur les 13 dernieres semaines (91 jours). Chaque jour est une cellule coloree selon le nombre de seances ce jour-la.

## Fichiers concernes
- `mobile/src/model/utils/streakHeatmapHelpers.ts` — NOUVEAU : calcul de la heatmap
- `mobile/src/screens/HomeScreen.tsx` — MODIFIER : ajouter carte heatmap
- `mobile/src/i18n/fr.ts` — MODIFIER : ajouter cles `home.heatmap.*`
- `mobile/src/i18n/en.ts` — MODIFIER : meme structure

## Implementation

### streakHeatmapHelpers.ts

```ts
export interface HeatmapDay {
  date: string          // 'YYYY-MM-DD'
  count: number         // nombre de seances ce jour
  intensity: 0 | 1 | 2 | 3  // 0 = repos, 1 = 1 seance, 2 = 2 seances, 3 = 3+
  isToday: boolean
}

export interface HeatmapResult {
  days: HeatmapDay[]    // 91 jours, du plus ancien au plus recent
  totalWorkouts: number
  activeDays: number
  currentStreak: number
  longestStreak: number
  weekLabels: string[]  // ['L', 'M', 'M', 'J', 'V', 'S', 'D']
}

/**
 * Genere la heatmap des 13 dernieres semaines.
 *
 * @param histories — toutes les seances (non abandonnees)
 *
 * Algorithme :
 * 1. Calculer la date de debut (91 jours avant aujourd'hui, aligne au lundi)
 * 2. Pour chaque jour dans la plage, compter le nombre de histories avec `started_at` ce jour
 * 3. Ignorer les histories avec `is_abandoned === true`
 * 4. intensity : 0 si count=0, 1 si count=1, 2 si count=2, 3 si count>=3
 * 5. Calculer les streaks (jours consecutifs avec au moins 1 seance)
 */
export function computeStreakHeatmap(
  histories: Array<{ startedAt: Date | number; isAbandoned: boolean }>,
): HeatmapResult
```

### Section UI dans HomeScreen

Ajouter **apres** la carte fatigue (ou recovery si existante). Consulter le fichier pour l'emplacement exact.

**N'afficher que si au moins 1 seance dans les 91 derniers jours.**

```tsx
{heatmapData && heatmapData.totalWorkouts > 0 && (
  <View style={styles.heatmapCard}>
    <View style={styles.heatmapHeader}>
      <Text style={styles.heatmapTitle}>{t.home.heatmap.title}</Text>
      <Text style={styles.heatmapSubtitle}>
        {heatmapData.activeDays} {t.home.heatmap.activeDays} | {t.home.heatmap.streak} {heatmapData.currentStreak}j
      </Text>
    </View>
    <View style={styles.heatmapGrid}>
      {/* 7 lignes (L-D) x 13 colonnes (semaines) */}
      {[0, 1, 2, 3, 4, 5, 6].map((dayOfWeek) => (
        <View key={dayOfWeek} style={styles.heatmapRow}>
          {Array.from({ length: 13 }, (_, weekIndex) => {
            const dayIndex = weekIndex * 7 + dayOfWeek
            const day = heatmapData.days[dayIndex]
            if (!day) return <View key={weekIndex} style={styles.heatmapCell} />
            return (
              <View
                key={weekIndex}
                style={[
                  styles.heatmapCell,
                  { backgroundColor: getHeatmapColor(day.intensity, colors) },
                  day.isToday && styles.heatmapToday,
                ]}
              />
            )
          })}
        </View>
      ))}
    </View>
    <View style={styles.heatmapLegend}>
      <Text style={styles.heatmapLegendText}>{t.home.heatmap.less}</Text>
      {[0, 1, 2, 3].map((intensity) => (
        <View
          key={intensity}
          style={[styles.heatmapLegendCell, { backgroundColor: getHeatmapColor(intensity as 0|1|2|3, colors) }]}
        />
      ))}
      <Text style={styles.heatmapLegendText}>{t.home.heatmap.more}</Text>
    </View>
  </View>
)}
```

**Fonction couleur :**
```ts
function getHeatmapColor(intensity: 0 | 1 | 2 | 3, colors: any): string {
  switch (intensity) {
    case 0: return colors.card       // repos = fond carte
    case 1: return '#2D6A4F'         // vert sombre
    case 2: return '#40916C'         // vert moyen
    case 3: return '#52B788'         // vert clair/vif
  }
}
```
