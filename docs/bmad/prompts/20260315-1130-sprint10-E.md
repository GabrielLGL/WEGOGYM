# Rapport — Comparaison Séance vs Précédente (#100) — Groupe E — 20260315-1130

## Objectif
Ajouter une section "Comparaison" dans `WorkoutSummarySheet` montrant la différence entre la séance qui vient de se terminer et la précédente séance avec les mêmes exercices. Affiche les deltas de volume, poids max, et nombre de sets pour chaque exercice commun. Éphémère — calculé à la volée.

## Implémentation

### sessionComparisonHelpers.ts

```ts
export interface ExerciseComparison {
  exerciseId: string
  exerciseName: string
  current: {
    volume: number          // Σ(weight × reps)
    maxWeight: number
    totalSets: number
    totalReps: number
  }
  previous: {
    volume: number
    maxWeight: number
    totalSets: number
    totalReps: number
    date: number            // timestamp de la séance précédente
  } | null
  deltas: {
    volume: number          // current - previous (kg)
    volumePercent: number   // ((current - previous) / previous) × 100
    maxWeight: number       // delta poids max
    maxWeightPercent: number
  } | null
}

export interface SessionComparison {
  exercises: ExerciseComparison[]
  overallVolumeDelta: number        // delta volume total
  overallVolumeDeltaPercent: number
  hasComparison: boolean            // au moins 1 exercice avec historique
}

/**
 * Compare la séance actuelle avec la dernière séance contenant les mêmes exercices.
 */
export function computeSessionComparison(
  currentSets: Array<{ weight: number; reps: number; exerciseId: string }>,
  previousSets: Array<{ weight: number; reps: number; exerciseId: string; createdAt: Date | number }>,
  exercises: Array<{ id: string; name: string }>,
  currentHistoryId: string,
): SessionComparison
```

**Algorithme :**
1. Grouper `currentSets` par exerciseId → calculer volume, maxWeight, totalSets, totalReps
2. Pour chaque exercice de la séance actuelle :
   - Trouver les sets précédents (même exerciseId, date plus ancienne, groupés par séance)
   - Prendre la séance la plus récente (pas la courante)
   - Calculer volume, maxWeight, totalSets, totalReps de cette séance précédente
3. Calculer les deltas
4. `hasComparison` = true si au moins 1 exercice a un `previous`

### Récupération des données historiques

Dans WorkoutSummarySheet, utiliser un `useEffect` + `useState` pour charger les sets historiques :

```ts
const [previousSets, setPreviousSets] = useState<WorkoutSet[]>([])

useEffect(() => {
  // Récupérer les exerciseIds de la séance actuelle
  const exerciseIds = [...new Set(sets.map(s => s.exerciseId))]
  if (exerciseIds.length === 0) return

  // Requête : tous les sets de ces exercices (pour trouver les séances précédentes)
  database.get<WorkoutSet>('sets')
    .query(
      Q.where('exercise_id', Q.oneOf(exerciseIds)),
      Q.on('histories', Q.and(
        Q.where('deleted_at', null),
        Q.or(Q.where('is_abandoned', null), Q.where('is_abandoned', false)),
      )),
      Q.sortBy('created_at', Q.desc),
      Q.take(500),  // limiter pour la perf
    )
    .fetch()
    .then(setPreviousSets)
}, [sets])
```

### Section UI dans WorkoutSummarySheet

Ajouter **après** la section intensité et son sectionDivider. Consulter le fichier pour l'emplacement exact.

**Ne PAS afficher si `!comparison.hasComparison`.**

```tsx
{comparison.hasComparison && (
  <>
    <View style={styles.comparisonSection}>
      <Text style={styles.comparisonTitle}>{t.sessionComparison.title}</Text>

      {/* Delta volume global */}
      <View style={styles.comparisonOverall}>
        <Text style={styles.comparisonOverallLabel}>{t.sessionComparison.totalVolume}</Text>
        <Text style={[
          styles.comparisonOverallDelta,
          { color: comparison.overallVolumeDelta >= 0 ? colors.primary : colors.danger }
        ]}>
          {comparison.overallVolumeDelta >= 0 ? '↑' : '↓'}
          {' '}{Math.abs(comparison.overallVolumeDeltaPercent).toFixed(1)}%
          {' '}({comparison.overallVolumeDelta >= 0 ? '+' : ''}{Math.round(comparison.overallVolumeDelta)} kg)
        </Text>
      </View>

      {/* Par exercice */}
      {comparison.exercises.filter(e => e.deltas).map((ex) => (
        <View key={ex.exerciseId} style={styles.comparisonExRow}>
          <Text style={styles.comparisonExName} numberOfLines={1}>{ex.exerciseName}</Text>
          <View style={styles.comparisonExDeltas}>
            <Text style={[
              styles.comparisonExDelta,
              { color: ex.deltas!.volume >= 0 ? colors.primary : colors.danger }
            ]}>
              {ex.deltas!.volume >= 0 ? '+' : ''}{Math.round(ex.deltas!.volume)} kg
            </Text>
            {ex.deltas!.maxWeight !== 0 && (
              <Text style={[
                styles.comparisonExDelta,
                { color: ex.deltas!.maxWeight >= 0 ? colors.primary : colors.danger }
              ]}>
                max {ex.deltas!.maxWeight >= 0 ? '+' : ''}{ex.deltas!.maxWeight} kg
              </Text>
            )}
          </View>
        </View>
      ))}
    </View>
    <View style={styles.sectionDivider} />
  </>
)}
```
