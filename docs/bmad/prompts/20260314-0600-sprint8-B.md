## Implémentation

### variantHelpers.ts

```ts
export interface VariantSuggestion {
  exercise: Exercise
  sharedMuscles: string[]      // muscles en commun avec l'exercice cible
  hasHistory: boolean          // l'utilisateur a déjà fait cet exercice
  similarityScore: number      // 0-1 : proportion de muscles en commun
}

/**
 * Trouve les 3 meilleures variantes pour un exercice donné.
 * Exclut l'exercice lui-même. Trie par score de similarité décroissant.
 * Sépare : déjà pratiquées en premier, puis nouvelles à découvrir.
 */
export function computeVariantSuggestions(
  targetExercise: Exercise,
  allExercises: Exercise[],
  usedExerciseIds: Set<string>,  // IDs des exercices avec au moins 1 set
  limit: number = 3,
): VariantSuggestion[]
```

**Algorithme :**
1. Exclure `targetExercise` de `allExercises`
2. Pour chaque exercice candidat :
   - `sharedMuscles` = intersection de muscles avec `targetExercise.muscles`
   - `similarityScore` = `sharedMuscles.length / Math.max(1, targetExercise.muscles.length)`
   - `hasHistory` = `usedExerciseIds.has(exercise.id)`
3. Filtrer : `sharedMuscles.length > 0`
4. Trier : `hasHistory` en premier, puis par `similarityScore` décroissant
5. Retourner les `limit` premiers

### Intégration dans ExerciseHistoryContent

**Imports à ajouter :**
```ts
import { computeVariantSuggestions } from '../model/utils/variantHelpers'
```

**useMemo dans ExerciseHistoryContent :**
```ts
// Construire Set des exercice IDs utilisés
const usedExerciseIds = useMemo(
  () => new Set(setsForExercise.map(s => s.exercise?.id).filter(Boolean) as string[]),
  [setsForExercise],
)

// Note: si `exercises` n'est pas dans les props actuelles, ajouter au withObservables
// Consulter le fichier pour la signature exacte

const variantSuggestions = useMemo(
  () => exercise && allExercises.length > 0
    ? computeVariantSuggestions(exercise, allExercises, usedExerciseIds)
    : [],
  [exercise, allExercises, usedExerciseIds],
)
```

**Section UI à ajouter après `prediction` en bas du ScrollView :**
```tsx
{variantSuggestions.length > 0 && (
  <View>
    <View style={styles.separator} />
    <Text style={[styles.sectionTitle, styles.historySectionTitle]}>
      {t.exerciseHistory.variants.title}
    </Text>
    {variantSuggestions.map((variant, index) => (
      <TouchableOpacity
        key={variant.exercise.id}
        style={styles.variantRow}
        onPress={() => {
          haptics.onPress()
          navigation.push('ExerciseHistory', { exerciseId: variant.exercise.id })
        }}
        activeOpacity={0.7}
      >
        {index > 0 && <View style={styles.separator} />}
        <View style={styles.variantContent}>
          <View style={styles.variantLeft}>
            <Text style={styles.variantName}>{variant.exercise.name}</Text>
            <Text style={styles.variantMuscles}>{variant.sharedMuscles.join(' · ')}</Text>
          </View>
          <View style={styles.variantRight}>
            {variant.hasHistory
              ? <Text style={styles.variantDone}>{t.exerciseHistory.variants.done}</Text>
              : <Text style={styles.variantNew}>{t.exerciseHistory.variants.discover}</Text>
            }
            <Ionicons name="chevron-forward" size={16} color={colors.textMuted} />
          </View>
        </View>
      </TouchableOpacity>
    ))}
  </View>
)}
```
