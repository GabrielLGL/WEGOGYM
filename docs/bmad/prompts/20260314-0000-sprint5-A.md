## Implémentation

### athleteClassHelpers.ts
```ts
export type AthleteClass = 'powerlifter' | 'bodybuilder' | 'complete' | 'polyvalent'

export interface AthleteClassResult {
  class: AthleteClass
  pushPct: number    // % du volume total en muscles push (0-100)
  pullPct: number    // % du volume total en muscles pull
  legsPct: number    // % du volume total en muscles legs
  corePct: number    // % du volume total en core
  pushPullRatio: number  // push / pull (1.0 = équilibré)
  upperLowerRatio: number // (push+pull) / legs
}

/**
 * Calcule la classe athlète depuis les sets et exercises.
 * Retourne null si moins de 20 sets au total.
 */
export function computeAthleteClass(
  sets: WorkoutSet[],
  exercises: Exercise[],
): AthleteClassResult | null
```

**Algorithme :**
1. Construire `Map<exerciseId, muscles[]>` depuis `exercises`
2. Pour chaque set, calculer `volume = weight * reps`
3. Pour chaque muscle de l'exercice associé, ajouter `volume / muscles.length` à la catégorie correspondante (push/pull/legs/core)
4. Si total < 20 sets → retourner `null`
5. Calculer les pourcentages
6. Déterminer la classe :
   - `powerlifter` : `legsPct > 30` ET `Math.abs(pushPct - pullPct) < 20`
   - `bodybuilder` : `pushPct + pullPct > 65` ET `pushPct > 30`
   - `complete` : `legsPct > 25` ET `pushPct > 20` ET `pullPct > 20`
   - `polyvalent` : défaut (aucune des conditions ci-dessus)
