# Rapport — Constellation de Progression (#66) — Groupe D — 20260314-0400

## Objectif
Créer un écran `StatsConstellationScreen` qui affiche chaque PR historique comme une étoile dans un espace SVG (x = date, y = 1RM estimé). Plus le PR est récent et élevé, plus l'étoile est grande et brillante. Une visualisation poétique de la progression. Accessible depuis StatsScreen.

## Implémentation

### Structure de données (dans useMemo)

```ts
interface ConstellationStar {
  x: number         // position normalisée 0-1 (date)
  y: number         // position normalisée 0-1 (1RM, inversé pour SVG — 0 = haut = élevé)
  size: number      // rayon en px (2-8 selon 1RM relatif)
  orm: number       // 1RM estimé
  exerciseName: string
  date: Date
  opacity: number   // 0.4-1.0 selon ancienneté (récent = opaque)
}
```

**Calcul :**
1. Filtrer `sets.filter(s => s.isPr === true)`
2. Si < 3 PRs → afficher empty state
3. Calculer `orm = Math.round(set.weight * (1 + set.reps / EPLEY_FORMULA_DIVISOR))`
4. Construire les étoiles :
   - `x` : normaliser `set.createdAt.getTime()` entre `minDate` et `maxDate` → 0 à 1
   - `y` : normaliser `orm` entre `minOrm` et `maxOrm` → **1 à 0** (inversé : haut = meilleur)
   - `size` : `2 + (orm / maxOrm) * 6` (entre 2 et 8)
   - `opacity` : `0.4 + (dateMs / maxDateMs) * 0.6` (récent = plus opaque)
5. Limiter à 200 étoiles max (les plus récentes)

### Rendu SVG

```tsx
const SVG_WIDTH = Dimensions.get('window').width - spacing.md * 2
const SVG_HEIGHT = 300

// Dans le composant
<Svg width={SVG_WIDTH} height={SVG_HEIGHT} style={styles.svg}>
  {/* Fond étoilé — petits points aléatoires décoratifs */}
  {backgroundStars.map((s, i) => (
    <Circle key={`bg-${i}`} cx={s.x} cy={s.y} r={0.8} fill={colors.textMuted} opacity={0.3} />
  ))}

  {/* Étoiles PRs */}
  {stars.map((star, i) => (
    <Circle
      key={i}
      cx={star.x * SVG_WIDTH}
      cy={star.y * SVG_HEIGHT}
      r={star.size}
      fill={colors.primary}
      opacity={star.opacity}
    />
  ))}
</Svg>
```

**Fond étoilé :** Générer 50 étoiles de fond avec positions fixes (seed déterministe basé sur index) :
```ts
const backgroundStars = useMemo(() =>
  Array.from({ length: 50 }, (_, i) => ({
    x: ((i * 137.5) % SVG_WIDTH),  // distribution pseudo-aléatoire
    y: ((i * 97.3) % SVG_HEIGHT),
  })), [SVG_WIDTH, SVG_HEIGHT])
```
