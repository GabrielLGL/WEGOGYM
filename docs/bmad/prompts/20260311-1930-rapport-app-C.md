# Rapport — Rapports hebdo/mensuels in-app — Groupe C — 20260311-1930

## Objectif
Créer l'écran `ReportDetailScreen` qui affiche le rapport complet (hebdo ou mensuel) avec navigation entre périodes passées. Intégrer dans la navigation.

## Fichiers concernés
- `mobile/src/screens/ReportDetailScreen.tsx` (NOUVEAU)
- `mobile/src/navigation/index.tsx` (ajouter le screen au navigator)

## Contexte technique

### Navigation existante
Fichier : `mobile/src/navigation/index.tsx`
- Utilise `createNativeStackNavigator` (React Navigation 7)
- Pattern : `<Stack.Screen name="ScreenName" component={ScreenComponent} />`
- Chaque écran reçoit `navigation` et `route` props
- Pour passer des params : `navigation.navigate('ReportDetail', { type: 'weekly', offset: 0 })`

### Données nécessaires (du Groupe A)
```typescript
import { getWeekPeriod, getMonthPeriod, computeReportSummary } from '../model/utils/statsHelpers'
```
- `getWeekPeriod(offset)` / `getMonthPeriod(offset)` → `ReportPeriod`
- `computeReportSummary(histories, sets, exercises, period, ctx)` → `ReportSummary`

### Pattern pour observer les données DB
```typescript
// HOC withObservables pour les données réactives
const enhance = withObservables([], ({ database }) => ({
  histories: database.get<History>('histories').query(
    Q.where('deleted_at', null),
    Q.where('is_abandoned', false)
  ).observe(),
  sets: database.get<Set>('sets').query().observe(),
  exercises: database.get<Exercise>('exercises').query().observe(),
}))
```

## Étapes

1. Ajouter les clés i18n dans `fr.ts` et `en.ts`

2. Créer `ReportDetailScreen.tsx` :
   - Route params : `{ type?: 'weekly' | 'monthly', offset?: number }` (défaut: weekly, 0)
   - State local : `periodOffset` (0 = courant, -1 = précédent, etc.)
   - State local : `reportType` ('weekly' | 'monthly') — toggle
   - Observer histories, sets, exercises via `withObservables`
   - `useMemo` : `prepareStatsContext` + `computeReportSummary`
   - Navigation prev/next : modifier `periodOffset`
   - Bouton "Suivant" désactivé si offset === 0 (période courante)

3. Intégrer dans la navigation :
   - Ajouter `ReportDetail` dans le Stack Navigator
   - Type de route params dans les types de navigation
