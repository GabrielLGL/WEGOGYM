## Implémentation

### 1. flashbackHelpers.ts
```ts
interface FlashbackData {
  label: string            // "Il y a 1 mois" ou "Il y a 3 mois"
  sessions: number         // séances dans la fenêtre de 7 jours
  volumeKg: number         // volume total (weight * reps)
  periodStart: number      // timestamp début de la fenêtre
  periodEnd: number        // timestamp fin de la fenêtre
}

// Retourne flashback 1 mois ET 3 mois si des données existent
// Fenêtre : ±3 jours autour de la même date (ex: aujourd'hui il y a 1 mois ±3 jours)
export function computeFlashback(
  histories: History[],
  sets: WorkoutSet[],
  monthsAgo: 1 | 3,
): FlashbackData | null
```

**Logique :**
- Calculer `targetDate = now - monthsAgo * 30 jours`
- Fenêtre : `[targetDate - 3 jours, targetDate + 3 jours]`
- Filtrer les histories dans cette fenêtre (`!isAbandoned && deletedAt === null`)
- Calculer sessions count et volume total des sets correspondants
- Retourner `null` si aucune history dans la fenêtre

### 2. FlashbackCard (composant inline dans HomeScreen)
Carte compacte affichant 2 colonnes :
```
┌──────────────────────────────────────────────────────┐
│ 📸 Flashback — Il y a 1 mois                        │
│                                                      │
│  Cette semaine  │  Il y a 1 mois                    │
│  3 séances      │  2 séances  ↑+1                   │
│  4 200 kg       │  3 100 kg   ↑+35%                 │
└──────────────────────────────────────────────────────┘
```

- Couleur delta positif : `colors.primary`, négatif : `colors.danger`
- N'afficher la carte QUE si au moins 1 des 2 flashbacks (1 mois ou 3 mois) retourne des données
- Priorité : montrer d'abord le 1 mois, puis le 3 mois en dessous si disponible
