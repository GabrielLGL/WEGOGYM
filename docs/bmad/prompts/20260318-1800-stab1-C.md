## Consignes pour chaque fichier test

### 1. setQualityHelpers.test.ts
Lire `setQualityHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne vide si aucun set

- Grade D pour sets tres inconsistants
- Drop sets detectes (poids decroissant dans une meme history)
- repConsistency dans [0, 100]
- Filtre par periode
- Minimum 5 sets requis par exercice

### 2. volumeRecordsHelpers.test.ts
Lire `volumeRecordsHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne vide si aucune donnee
- Record session = max volume d'une seule seance
- Record semaine = max volume sur 7 jours (lundi-dimanche)
- Record mois = max volume sur un mois calendaire
- isNewRecord = true si volume en cours > record
- recentTrend correct (up/down/stable)

### 3. streakMilestonesHelpers.test.ts
Lire `streakMilestonesHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne streak 0 si aucune history
- Streak correct avec 1 jour de gap tolere
- Milestones atteints marques reached=true
- nextMilestone = premier non-atteint
- progressToNext dans [0, 100]
- 8 milestones predefinis (3, 7, 14, 30, 60, 100, 200, 365)

### 4. workoutSummaryHelpers.test.ts
Lire `workoutSummaryHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne null si aucune history
- Derniere seance selectionnee (tri par startedAt)

- Duration en minutes

### 5. restDaySuggestionsHelpers.test.ts
Lire `restDaySuggestionsHelpers.ts`, identifier les fonctions exportees.

### 6. exerciseMasteryHelpers.test.ts
