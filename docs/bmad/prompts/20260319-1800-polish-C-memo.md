## Contexte technique

### Pattern a suivre
L'app a deja UN exemple de React.memo bien fait dans `ExercisesScreen.tsx` :

```tsx
// Composant extrait et memoize
const ExerciseItem = React.memo(function ExerciseItem({ ... }: ExerciseItemProps) {
  return (...)
}, (prev, next) => {
  // Custom comparator optionnel
  return prev.item.id === next.item.id && prev.selected === next.selected
})
```

### Regles
1. **Extraire** le contenu du renderItem dans un composant nomme (si c'est inline)
2. **Wrapper** avec `React.memo()`
3. **Typer** les props avec une interface
4. **Garder** le `useCallback` sur le renderItem dans le parent
5. **NE PAS** ajouter de custom comparator sauf si necessaire (le shallow compare par defaut suffit dans la plupart des cas)
6. Les callbacks passes en props (onPress, etc.) doivent DEJA etre des `useCallback` dans le parent — sinon React.memo ne sert a rien

### Patterns obligatoires (CLAUDE.md)
- `useColors()` hook pour les couleurs — ATTENTION : si le composant memoize utilise `useColors()` en interne, les couleurs changent quand le theme change → re-render automatique, c'est correct
- TypeScript strict, pas de `any`
- Composants fonctionnels uniquement

### Exemple d'implementation

**Avant :**
```tsx
// Dans le screen
const renderCard = useCallback(({ item }: { item: ActivityItem }) => (
  <TouchableOpacity style={styles.card}>
    <Text>{item.title}</Text>
    <Text>{item.subtitle}</Text>
  </TouchableOpacity>
), [styles])
```

**Apres :**
```tsx
// Composant extrait (dans le meme fichier, au-dessus du screen)
interface ActivityCardProps {
  item: ActivityItem
  onPress: (id: string) => void
}

const ActivityCard = React.memo(function ActivityCard({ item, onPress }: ActivityCardProps) {
  const colors = useColors()
  const styles = useCardStyles(colors)
  return (
    <TouchableOpacity style={styles.card} onPress={() => onPress(item.id)}>
      <Text style={styles.title}>{item.title}</Text>
      <Text style={styles.subtitle}>{item.subtitle}</Text>
    </TouchableOpacity>
  )
})

// Dans le screen
const renderCard = useCallback(({ item }: { item: ActivityItem }) => (
  <ActivityCard item={item} onPress={handlePress} />
), [handlePress])
```
