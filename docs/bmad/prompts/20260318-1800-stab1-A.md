## Consignes pour chaque fichier test

Lire `repMaxHelpers.ts`, identifier les fonctions exportees (ex: `estimateOneRepMax`, `computeRepMaxHistory`).

- Retourne null/0 si aucun set
- Calcul 1RM correct pour un set typique (100kg x 10 → ~133kg Epley)
- Historique 1RM trie par date
- Gere les sets avec reps=1 (1RM direct)
- Gere les sets avec poids=0

Lire `volumeDistributionHelpers.ts`, identifier les fonctions exportees.

- Filtre par periode (30j, 90j, tout)
- Pourcentages somment a 100%
- Gere les exercices multi-muscles

### 3. monthlyProgressHelpers.test.ts
Lire `monthlyProgressHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne vide si aucune donnee
- Groupement correct par mois
- Calcul de progression mois sur mois
- Gere un seul mois de donnees
- Volume et sessions comptes correctement

### 4. streakHeatmapHelpers.test.ts
Lire `streakHeatmapHelpers.ts`, identifier les fonctions exportees.
Tests :
- Retourne grille vide si aucune history
- 13 semaines generees

### 5. restTimeAnalysisHelpers.test.ts
Lire `restTimeAnalysisHelpers.ts`, identifier les fonctions exportees.
Tests :
