## Implémentation

### exerciseAlternativesHelpers.ts

```ts
export interface AlternativeExercise {
  exerciseId: string
  exerciseName: string
  sharedMuscles: string[]    // muscles en commun
  matchScore: number         // 0-1 (1 = tous les muscles en commun)
  totalSets: number          // nombre de sets historiques
  lastUsed: number | null    // timestamp dernière utilisation
}

/**
 * Trouve des exercices alternatifs basés sur les muscles en commun.
 * Retourne les 5 meilleures alternatives (exercices pratiqués uniquement).
 */
export function findAlternatives(
  currentExercise: { id: string; name: string; muscles: string[] },
  allExercises: Array<{ id: string; name: string; muscles: string[] }>,
  allSets: Array<{ exerciseId: string; createdAt: Date | number }>,
  maxResults?: number,  // défaut: 5
): AlternativeExercise[]
```

**Algorithme :**
1. Filtrer les exercices : exclure l'exercice courant
2. Pour chaque exercice restant :
   - Calculer `sharedMuscles` = intersection avec les muscles de l'exercice courant
   - Si 0 muscle en commun → exclure
   - `matchScore` = sharedMuscles.length / max(currentExercise.muscles.length, exercise.muscles.length)
3. Filtrer : ne garder que les exercices avec au moins 1 set en DB (déjà pratiqués)
4. Pour chaque : compter `totalSets` et trouver `lastUsed` (max createdAt)
5. Trier par `matchScore` desc, puis par `totalSets` desc
6. Retourner les `maxResults` premiers
