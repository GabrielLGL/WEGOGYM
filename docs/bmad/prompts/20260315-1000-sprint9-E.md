## Implémentation

### trainingCalendarHelpers.ts

```ts
export interface CalendarDay {
  date: number            // timestamp début de journée
  setsCount: number       // nombre de sets ce jour
  volume: number          // volume total (kg)
  intensity: 0 | 1 | 2 | 3 | 4  // 0=repos, 1-4=quartiles d'intensité
  isToday: boolean
}

export interface CalendarWeek {
  days: CalendarDay[]     // 7 jours (lundi → dimanche)
  weekNumber: number
}

/**
 * Génère les données de la heatmap pour les N dernières semaines.
 */
export function computeTrainingCalendar(
  histories: Array<{ createdAt: Date | number; deletedAt: Date | null; isAbandoned: boolean }>,
  sets: Array<{ createdAt: Date | number }>,
  weeks?: number,  // défaut: 12
): CalendarWeek[]
```

**Algorithme :**
1. Calculer la date de début = aujourd'hui - (weeks × 7) jours, aligné au lundi
2. Créer un tableau de jours de cette date à aujourd'hui
3. Pour chaque jour : compter les sets (filtrer les histories valides — non supprimées, non abandonnées)
4. Calculer les quartiles de setsCount sur les jours non-repos :
   - Q1 (25%) → intensity 1, Q2 (50%) → 2, Q3 (75%) → 3, max → 4
5. Jours à 0 sets → intensity 0
6. Grouper par semaine (lundi → dimanche)

Ajouter **après** la section athlete class / avant la section statistiques rapides. Consulter le fichier pour l'emplacement exact — chercher un commentaire comme `{/* ── Stats rapides ── */}` ou la section qui suit le header XP/level.

```tsx
{/* ── Calendrier Heatmap ── */}
<View style={styles.heatmapSection}>
  <Text style={styles.heatmapTitle}>{t.home.heatmap.title}</Text>
  <View style={styles.heatmapGrid}>
    {calendarWeeks.map((week, wi) => (
      <View key={wi} style={styles.heatmapColumn}>
        {week.days.map((day, di) => (
          <View
            key={di}
            style={[
              styles.heatmapCell,
              {
                backgroundColor: day.intensity === 0
                  ? colors.cardSecondary
                  : day.intensity === 1
                    ? colors.primary + '33'
                    : day.intensity === 2
                      ? colors.primary + '66'
                      : day.intensity === 3
                        ? colors.primary + '99'
                        : colors.primary,
              },
              day.isToday && styles.heatmapCellToday,
            ]}
          />
        ))}
      </View>
    ))}
  </View>
  <View style={styles.heatmapLegend}>
    <Text style={styles.heatmapLegendText}>{t.home.heatmap.less}</Text>
    {[0, 1, 2, 3, 4].map(i => (
      <View
        key={i}
        style={[
          styles.heatmapLegendCell,
          {
            backgroundColor: i === 0
              ? colors.cardSecondary
              : i === 1
                ? colors.primary + '33'
                : i === 2
                  ? colors.primary + '66'
                  : i === 3
                    ? colors.primary + '99'
                    : colors.primary,
          },
        ]}
      />
    ))}
    <Text style={styles.heatmapLegendText}>{t.home.heatmap.more}</Text>
  </View>
</View>
```
