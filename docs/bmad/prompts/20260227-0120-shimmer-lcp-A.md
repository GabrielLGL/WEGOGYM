# Rapport — shimmer-text LCP — Groupe A — 20260227-0120

## Objectif
Optimiser l'animation .shimmer-text dans web/src/app/globals.css pour réduire son impact LCP.

## Fichiers concernés
- web/src/app/globals.css

## Contexte technique
Projet web Next.js (web/). Le fichier globals.css contient les classes CSS globales incluant l'animation shimmer-text. Aucune dépendance React/TS — changement CSS pur.

## Étapes
1. Dans .shimmer-text : supprimer `animation: shimmer 4s linear infinite;`
2. Dans .shimmer-text : ajouter `will-change: background-position;`
3. Après le bloc .shimmer-text : ajouter un @media (prefers-reduced-motion: no-preference) contenant `.shimmer-text { animation: shimmer 6s linear infinite; }`

## Contraintes
- Ne pas modifier le bloc @keyframes shimmer
- Ne pas modifier le bloc @media (prefers-reduced-motion: reduce) existant
- Ne toucher aucun autre sélecteur

## Critères de validation
- .shimmer-text ne contient plus `animation:` en dehors du media query
- will-change: background-position présent dans .shimmer-text
- @media (prefers-reduced-motion: no-preference) présent avec shimmer 6s
- Le bloc reduce existant est intact

## Dépendances
Aucune.
