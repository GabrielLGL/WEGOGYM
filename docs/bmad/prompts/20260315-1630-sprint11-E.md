# Rapport — Niveaux de Maîtrise des Exercices (#105) — Groupe E — 20260315-1630

## Objectif
Ajouter un système de "niveau de maîtrise" par exercice dans `ExercisesScreen`. Chaque exercice affiche un badge de maîtrise (1-5 étoiles) basé sur le nombre de sets réalisés, la variété des rep ranges, et la progression du poids. Le badge est affiché directement dans la liste d'exercices. Éphémère — calculé à la volée.

## Implémentation

### exerciseMasteryHelpers.ts

```ts
export interface ExerciseMastery {
  exerciseId: string
  level: 0 | 1 | 2 | 3 | 4 | 5   // 0 = non pratiqué
  totalSets: number
  distinctRepRanges: number         // nombre de ranges différents (1-5, 6-8, 9-12, 13-15, 16+)
  hasProgression: boolean           // le poids a augmenté au fil du temps
  score: number                     // score brut (0-100) pour le tri
}

/**
 * Calcule le niveau de maîtrise pour chaque exercice.
 */
export function computeExerciseMastery(
  exercises: Array<{ id: string }>,
  sets: Array<{ weight: number; reps: number; exerciseId: string; createdAt: Date | number }>,
): Map<string, ExerciseMastery>
```

**Algorithme :**
Pour chaque exercice :
1. Compter `totalSets` pour cet exercice
2. Classifier les reps en ranges : [1-5], [6-8], [9-12], [13-15], [16+]
   - `distinctRepRanges` = nombre de ranges utilisés
3. Vérifier `hasProgression` :
   - Diviser les sets en 2 moitiés chronologiques
   - Si le poids moyen de la 2e moitié > 1re moitié → true
4. Calculer le **score** (0-100) :
   - `setsScore` = min(40, totalSets × 0.5)
   - `varietyScore` = min(30, distinctRepRanges × 6)
   - `progressionScore` = hasProgression ? 30 : 0
   - `score` = setsScore + varietyScore + progressionScore
5. **Level** :
   - score = 0 → level 0 (non pratiqué)
   - score < 20 → level 1
   - score < 40 → level 2
   - score < 60 → level 3
   - score < 80 → level 4
   - score ≥ 80 → level 5

### Modification de ExercisesScreen

Dans la liste d'exercices (FlatList ou ScrollView), ajouter le badge de maîtrise à chaque item.

Lire le fichier pour trouver le composant qui rend chaque exercice dans la liste. Ajouter à côté du nom de l'exercice :

```tsx
{/* Badge maîtrise */}
{mastery && mastery.level > 0 && (
  <View style={styles.masteryBadge}>
    {Array.from({ length: 5 }, (_, i) => (
      <Ionicons
        key={i}
        name={i < mastery.level ? 'star' : 'star-outline'}
        size={10}
        color={i < mastery.level ? getMasteryColor(mastery.level, colors) : colors.textMuted}
      />
    ))}
  </View>
)}
```

**Fonction couleur :**
```ts
function getMasteryColor(level: number, colors: any): string {
  if (level <= 2) return colors.textSecondary
  if (level <= 3) return '#F59E0B'       // jaune
  if (level === 4) return colors.primary
  return '#8B5CF6'                        // violet pour level 5
}
```

**Styles :**
```ts
masteryBadge: {
  flexDirection: 'row',
  gap: 1,
  marginLeft: spacing.xs,
},
```

Le calcul de mastery doit être fait une seule fois via `useMemo` :
```tsx
const masteryMap = useMemo(
  () => computeExerciseMastery(exercises, sets),
  [exercises, sets],
)
```
