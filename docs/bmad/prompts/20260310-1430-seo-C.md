# Rapport — SEO actions manuelles — Groupe C — 20260310-1430

## Objectif
Actions manuelles (non-code) indispensables pour que Google indexe et référence kore-app.net.

## PRIORITÉ #1 — Google Search Console (CRITIQUE)

Sans ça, rien d'autre ne sert.

### Étapes :
1. Va sur https://search.google.com/search-console
2. Clique "Ajouter une propriété"
3. Choisis "Préfixe d'URL" → entre `https://kore-app.net`
4. Vérifie la propriété :
   - **Méthode recommandée** : ajouter un enregistrement TXT DNS dans Cloudflare
   - Ou : télécharger le fichier HTML de vérification et le mettre dans `web/public/`
5. Une fois vérifié :
   - Va dans "Sitemaps" → soumets `https://kore-app.net/sitemap.xml`
   - Va dans "Inspection d'URL" → entre `https://kore-app.net` → "Demander l'indexation"
   - Fais pareil pour `/contact`, `/privacy`, `/cgu`, `/mentions-legales`

### Résultat attendu
- Google crawle et indexe le site sous 2-7 jours
- Le sitemap apparaît comme "Réussi" dans la Search Console
