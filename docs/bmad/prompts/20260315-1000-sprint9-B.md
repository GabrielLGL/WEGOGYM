## Implémentation

### progressiveOverloadHelpers.ts

```ts
export interface OverloadTrend {
  metric: 'weight' | 'volume'
  trend: 'up' | 'down' | 'stable'
  percentChange: number       // ex: +12.5 ou -3.2
  lastSessions: number        // nombre de séances analysées
  dataPoints: Array<{
    date: number              // timestamp
    value: number             // poids max ou volume total
  }>
}

/**
 * Analyse la tendance de surcharge progressive sur les N dernières séances.
 * Calcule à la fois la tendance du poids max et du volume par séance.
 */
export function computeOverloadTrend(
  sets: Array<{ weight: number; reps: number; createdAt: Date | number }>,
  windowSize?: number,  // défaut: 5 séances
): { weightTrend: OverloadTrend; volumeTrend: OverloadTrend }
```

**Algorithme :**
1. Grouper les sets par date (même jour = même séance)
2. Pour chaque séance, calculer :
   - `maxWeight` = poids le plus élevé
   - `sessionVolume` = Σ(weight × reps)
3. Prendre les `windowSize` dernières séances (défaut 5)
4. Calculer la régression linéaire simple (slope) sur les valeurs
5. `percentChange` = ((dernière - première) / première) × 100
6. `trend` : slope > 2% → 'up', slope < -2% → 'down', sinon 'stable'

### Section UI dans ExerciseHistoryScreen

Ajouter **après** le graphique existant (LineChart), avant la section "prediction" :

```tsx
{weightTrend.lastSessions >= 3 && (
  <View style={styles.overloadSection}>
    <Text style={styles.overloadTitle}>{t.exerciseHistory.overload.title}</Text>

    {/* Weight trend */}
    <View style={styles.overloadRow}>
      <Ionicons name="barbell-outline" size={16} color={colors.textSecondary} />
      <Text style={styles.overloadLabel}>{t.exerciseHistory.overload.maxWeight}</Text>
      <View style={styles.overloadTrendBadge}>
        <Text style={[
          styles.overloadTrendText,
          { color: weightTrend.trend === 'up' ? colors.primary
            : weightTrend.trend === 'down' ? colors.danger
            : colors.textSecondary }
        ]}>
          {weightTrend.trend === 'up' ? '↑' : weightTrend.trend === 'down' ? '↓' : '→'}
          {' '}{Math.abs(weightTrend.percentChange).toFixed(1)}%
        </Text>
      </View>
    </View>

    {/* Volume trend */}
    <View style={styles.overloadRow}>
      <Ionicons name="trending-up-outline" size={16} color={colors.textSecondary} />
      <Text style={styles.overloadLabel}>{t.exerciseHistory.overload.volume}</Text>
      <View style={styles.overloadTrendBadge}>
        <Text style={[
          styles.overloadTrendText,
          { color: volumeTrend.trend === 'up' ? colors.primary
            : volumeTrend.trend === 'down' ? colors.danger
            : colors.textSecondary }
        ]}>
          {volumeTrend.trend === 'up' ? '↑' : volumeTrend.trend === 'down' ? '↓' : '→'}
          {' '}{Math.abs(volumeTrend.percentChange).toFixed(1)}%
        </Text>
      </View>
    </View>

    <Text style={styles.overloadDisclaimer}>
      {t.exerciseHistory.overload.disclaimer.replace('{n}', String(weightTrend.lastSessions))}
    </Text>
  </View>
)}
```
