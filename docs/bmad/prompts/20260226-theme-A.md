# Rapport — Reliefs d'ombre — Groupe A — 20260226

## Objectif
Augmenter significativement le contraste et la profondeur des ombres neumorphiques
dans `neuShadow` pour que le relief soit clairement visible sur Android ET iOS.

## Problème actuel
- `shadowColor: '#16181d'` est trop proche du fond `#21242b` (delta faible)
- `borderColor: '#2c3039'` trop subtil (borderWidth 0.5 invisible)
- Android elevation 6 insuffisant en dark mode (ombres grises quasi-invisibles)

## Fichiers concernés
- `mobile/src/theme/index.ts` UNIQUEMENT

## Changements exacts dans neuShadow

Remplacer tout le bloc `export const neuShadow` par :

```ts
export const neuShadow = {
  elevated: {
    ...Platform.select({
      ios: {
        shadowColor: '#0a0c10',       // plus sombre que #16181d
        shadowOffset: { width: 8, height: 8 },   // était 6,6
        shadowOpacity: 0.9,
        shadowRadius: 16,             // était 12
      },
      android: { elevation: 10 },    // était 6
    }),
    borderWidth: 1,                  // était 0.5
    borderColor: '#353b47',          // était #2c3039, plus contrasté
  },
  elevatedSm: {
    ...Platform.select({
      ios: {
        shadowColor: '#0a0c10',
        shadowOffset: { width: 4, height: 4 },   // était 3,3
        shadowOpacity: 0.8,
        shadowRadius: 8,              // était 6
      },
      android: { elevation: 5 },    // était 3
    }),
    borderWidth: 1,                  // était 0.5
    borderColor: '#353b47',
  },
  pressed: {
    ...Platform.select({
      ios: {
        shadowColor: '#0a0c10',
        shadowOffset: { width: 1, height: 1 },
        shadowOpacity: 0.4,
        shadowRadius: 3,
      },
      android: { elevation: 1 },
    }),
    borderWidth: 1,                  // était 0.5
    borderColor: '#1a1d22',          // était #16181d, plus sombre pour effet enfoncé
  },
}
```

## Contraintes
- Ne pas toucher `colors`, `spacing`, `fontSize`, `borderRadius`, `intensityColors`
- Les valeurs en dur dans neuShadow sont intentionnelles (évite import circulaire)
- Le `borderColor` du `pressed` doit être PLUS SOMBRE que le background pour l'effet concave

## Critères de validation
- `npx tsc --noEmit` → zéro erreur
- Visuellement : sur Android, les cards doivent avoir un liseré visible
- Visuellement : sur iOS, l'ombre portée doit être nettement perceptible
