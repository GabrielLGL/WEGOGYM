## Implémentation

### fatigueIndexHelpers.ts

```ts
export type FatigueZone = 'recovery' | 'optimal' | 'reaching' | 'overreaching'

export interface FatigueResult {
  index: number              // 0-100 (0=repos total, 100=surmenage max)
  zone: FatigueZone
  weeklyVolume: number       // volume des 7 derniers jours
  avgWeeklyVolume: number    // moyenne hebdo sur 8 semaines
  ratio: number              // weeklyVolume / avgWeeklyVolume
  sessionsThisWeek: number
  avgSessionsPerWeek: number
  recommendation: string     // clé i18n
}

/**
 * Calcule l'indice de fatigue basé sur le ratio volume récent vs moyenne.
 * Utilise le modèle ACWR (Acute:Chronic Workload Ratio) simplifié.
 */
export function computeFatigueIndex(
  sets: Array<{ weight: number; reps: number; createdAt: Date | number }>,
  histories: Array<{ createdAt: Date | number; deletedAt: Date | null; isAbandoned: boolean }>,
): FatigueResult
```

**Algorithme :**
1. Calculer le volume des 7 derniers jours (acute load)
2. Calculer le volume moyen hebdo sur les 8 dernières semaines (chronic load)
3. Ratio ACWR = acute / chronic (si chronic = 0, ratio = 0)
4. **Index** = clamp(ratio × 50, 0, 100) — ratio 1.0 = index 50 (optimal)
5. **Zone** :
   - ratio < 0.5 → 'recovery' (sous-entraînement)
   - 0.5 ≤ ratio < 1.3 → 'optimal'
   - 1.3 ≤ ratio < 1.5 → 'reaching' (attention)
   - ratio ≥ 1.5 → 'overreaching' (surmenage)
6. **recommendation** : clé i18n correspondant à la zone
7. Compter les séances cette semaine vs moyenne

### Section UI dans HomeScreen

Ajouter **après** la section heatmap calendrier. Consulter le fichier pour l'emplacement exact.

**Ne PAS afficher la carte si aucune donnée (0 sets).**

```tsx
{fatigueResult && fatigueResult.weeklyVolume > 0 && (
  <View style={styles.fatigueCard}>
    <View style={styles.fatigueHeader}>
      <Ionicons
        name={fatigueResult.zone === 'overreaching' ? 'warning-outline'
          : fatigueResult.zone === 'reaching' ? 'alert-circle-outline'
          : fatigueResult.zone === 'recovery' ? 'bed-outline'
          : 'checkmark-circle-outline'}
        size={20}
        color={fatigueColor}
      />
      <Text style={[styles.fatigueTitle, { color: fatigueColor }]}>
        {t.home.fatigue.zones[fatigueResult.zone]}
      </Text>
    </View>
    <View style={styles.fatigueBarBg}>
      <View style={[
        styles.fatigueBarFill,
        { width: `${fatigueResult.index}%`, backgroundColor: fatigueColor }
      ]} />
      {/* Marker zone optimale (50%) */}
      <View style={[styles.fatigueMarker, { left: '50%' }]} />
    </View>
    <Text style={styles.fatigueStats}>
      {t.home.fatigue.thisWeek}: {Math.round(fatigueResult.weeklyVolume)} kg
      {'  •  '}
      {t.home.fatigue.average}: {Math.round(fatigueResult.avgWeeklyVolume)} kg
    </Text>
    <Text style={styles.fatigueRecommendation}>
      {t.home.fatigue.recommendations[fatigueResult.zone]}
    </Text>
  </View>
)}
```

**Couleurs par zone :**
- recovery: `colors.textMuted`
- optimal: `colors.primary`
- reaching: `'#F59E0B'`
- overreaching: `colors.danger`
