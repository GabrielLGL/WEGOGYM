## Implémentation

### sessionIntensityHelpers.ts

```ts
export interface IntensityResult {
  score: number           // 0-100
  label: string           // clé i18n : 'light' | 'moderate' | 'intense' | 'extreme'
  color: string           // couleur dynamique selon le score
  breakdown: {
    volumeScore: number   // 0-33 : volume total (poids × reps)
    prScore: number       // 0-33 : nombre de PRs dans la séance
    effortScore: number   // 0-34 : nombre de sets à RPE élevé (poids ≥ 80% du max historique)
  }
}

/**
 * Calcule le score d'intensité d'une séance.
 * @param sessionSets - les sets de la séance actuelle
 * @param allSets - tous les sets historiques (pour les max)
 */
export function computeSessionIntensity(
  sessionSets: Array<{ weight: number; reps: number; isPr: boolean; exerciseId: string }>,
  allSets: Array<{ weight: number; exerciseId: string }>,
): IntensityResult
```

**Algorithme :**
1. **volumeScore** (0-33) : volume total = Σ(weight × reps). Clamp : 0 si < 500kg, 33 si > 10000kg, linéaire entre les deux.
2. **prScore** (0-33) : nombre de PRs. 0 PR = 0, 1 PR = 11, 2 PRs = 22, 3+ PRs = 33.
3. **effortScore** (0-34) : pour chaque exercice, calculer le max historique. Compter les sets où weight ≥ 80% du max. Ratio = count / totalSets. Score = ratio × 34.
4. **score** = volumeScore + prScore + effortScore (0-100)
5. **label** : < 30 → 'light', < 55 → 'moderate', < 80 → 'intense', ≥ 80 → 'extreme'
6. **color** : light → `colors.textSecondary`, moderate → `colors.primary`, intense → `'#F59E0B'`, extreme → `colors.danger`
