## Contexte technique

### i18n
- Les labels d'accessibilite doivent etre traduits via `useLanguage()` → `{ t }`
- Ajouter une section `accessibility` dans `mobile/src/i18n/fr.ts` et `en.ts`
- **IMPORTANT** : `fr.ts` ne doit PAS avoir `as const`

```typescript
// Exemple de section a ajouter dans fr.ts et en.ts
accessibility: {
  settings: 'Parametres',
  startWorkout: 'Demarrer une seance',
  resumeWorkout: 'Reprendre la seance en cours',
  programs: 'Programmes',
  exercises: 'Exercices',
  calendar: 'Calendrier',
  duration: 'Duree',
  closeModal: 'Fermer',
  confirm: 'Confirmer',
  cancel: 'Annuler',
  darkMode: 'Mode sombre',
  language: 'Langue',
  // etc.
}
```

### Patterns d'accessibilite React Native

```tsx
// Bouton
<TouchableOpacity
  accessibilityRole="button"
  accessibilityLabel={t.accessibility.startWorkout}
  accessibilityHint="Double-tap pour demarrer"
>

// Switch
<Switch
  accessibilityRole="switch"
  accessibilityLabel={t.accessibility.darkMode}
  accessibilityState={{ checked: isDarkMode }}
/>

// Input
<TextInput
  accessibilityLabel={t.accessibility.nameInput}
  accessibilityHint="Entrez votre prenom"
/>

// Dialog
<View accessibilityRole="alert" accessibilityLabel={title}>

// Section header
<View accessibilityRole="header">
  <Text>{sectionTitle}</Text>
</View>
```

### Regles
- `accessibilityRole` : toujours sur les elements interactifs (button, switch, link, header, alert)
- `accessibilityLabel` : description courte de l'element (ce que c'est)
- `accessibilityHint` : optionnel, ce qui va se passer (seulement si pas evident)
- `accessibilityState` : pour les toggles (`{ checked }`, `{ selected }`, `{ disabled }`)
- Les labels doivent etre i18n (pas hardcodes en francais)

## Implementation par fichier

### Button.tsx
```tsx
// Ajouter aux props :
accessibilityLabel?: string
accessibilityHint?: string

// Dans le Pressable :
accessibilityRole="button"
accessibilityLabel={accessibilityLabel ?? children} // fallback sur le texte du bouton
accessibilityHint={accessibilityHint}
accessibilityState={{ disabled: !!disabled }}
```

### AlertDialog.tsx
```tsx
// Container du dialog :
accessibilityRole="alert"
accessibilityLabel={title}

// Bouton confirmer :
accessibilityRole="button"
accessibilityLabel={confirmText}

// Bouton annuler :
accessibilityRole="button"
accessibilityLabel={cancelText}
```

### BottomSheet.tsx
```tsx
// Container :
accessibilityRole="none" // ou "dialog" si supporte
accessibilityLabel={title}

// Overlay (zone de dismiss) :
accessibilityRole="button"
accessibilityLabel={t.accessibility.closeModal}
accessibilityHint="Double-tap pour fermer"
```

### HomeHeaderCard.tsx
```tsx
// Bouton settings :
accessibilityRole="button"
accessibilityLabel={t.accessibility.settings}
```

### HomeHeroAction.tsx
```tsx
// CTA principal (hero card) :
accessibilityRole="button"
accessibilityLabel={activeWorkout ? t.accessibility.resumeWorkout : t.accessibility.startWorkout}

// Shortcut pills :
accessibilityRole="button"
accessibilityLabel={s.label} // deja le nom de la destination
```

### HomeStatusStrip.tsx
```tsx
// Chaque chip :
accessibilityRole="button"
accessibilityLabel={`${chip.label} ${chip.icon === '🔥' ? 'streak' : ''}`}
```

### SettingsScreen.tsx
```tsx
// Name input :
accessibilityLabel={t.settings.name}

// Dark mode switch :
accessibilityRole="switch"
accessibilityLabel={t.accessibility.darkMode}
accessibilityState={{ checked: theme === 'dark' }}

// Language buttons :
accessibilityRole="button"
accessibilityLabel="Francais" / "English"
accessibilityState={{ selected: language === 'fr' }}

// Timer switch :
accessibilityRole="switch"
accessibilityLabel={t.settings.timer}

// Level/Goal toggles :
accessibilityRole="button"
accessibilityHint="Double-tap pour developper"
```
