# Rapport — blobs-perf — Groupe A — 20260227-0900

## Objectif
Optimiser `BackgroundBlobs.tsx` pour réduire le coût de repaint :
remplacer `blur-[80px]` par `blur-[60px]`, ajouter `willChange: 'transform'` sur chaque blob,
réduire l'opacité de `opacity-40` à `opacity-25`, et remplacer `fixed` par `absolute` sur le container.

## Fichiers concernés
- `web/src/components/BackgroundBlobs.tsx`

## Contexte technique
Le composant rend deux blobs animés (animation custom `floatBlob`) en fond de page.
Actuellement le container est `position: fixed` ce qui force un repaint lors du scroll.
Le passage à `absolute` + `will-change: transform` sur les blobs les promeut en layers GPU.
Les parents (`page.tsx`, `privacy/page.tsx`) ont déjà `relative` — le Groupe B ajoute `overflow-hidden`.

## Étapes

1. Ouvrir `web/src/components/BackgroundBlobs.tsx`
2. Container div (premier `<div>`) :
   - Remplacer `fixed` par `absolute`
   - Le reste reste identique : `inset-0 z-0 pointer-events-none overflow-hidden`
3. Blob 1 (premier blob `<div>`) :
   - Remplacer `blur-[80px]` par `blur-[60px]`
   - Remplacer `opacity-40` par `opacity-25`
   - Dans `style={{...}}`, ajouter `willChange: 'transform'`
4. Blob 2 (deuxième blob `<div>`) :
   - Remplacer `blur-[80px]` par `blur-[60px]`
   - Remplacer `opacity-40` par `opacity-25`
   - Dans `style={{...}}`, ajouter `willChange: 'transform'`
5. Sauvegarder
