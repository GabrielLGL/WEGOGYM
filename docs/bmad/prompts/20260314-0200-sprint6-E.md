# Rapport — Checklist Échauffement (#62) — Groupe E — 20260314-0200

## Objectif
Ajouter un bouton "Échauffement" dans le header de `WorkoutScreen` qui ouvre une BottomSheet avec une checklist d'échauffement adaptée aux exercices de la séance. Chaque item est cochable (état local, pas de DB). La checklist suggère des exercices de mobilité et des séries légères selon les groupes musculaires détectés.

**Générer les suggestions depuis les muscles :**
1. Collecter tous les muscles uniques des exercises de la séance
2. Pour chaque muscle, récupérer ses suggestions
3. Dédupliquer (un même exercice peut appartenir à plusieurs muscles)
4. Limiter à 8 suggestions max
5. Si aucun muscle → utiliser des suggestions génériques basiques

### WarmupChecklistSheet.tsx

```tsx
interface WarmupChecklistSheetProps {
  visible: boolean
  onClose: () => void
  muscles: string[]   // muscles uniques de la séance
}

export function WarmupChecklistSheet({ visible, onClose, muscles }: WarmupChecklistSheetProps)
```

**Rendu interne :**
```tsx
<BottomSheet visible={visible} onClose={onClose} title={t.warmup.title}>
  <ScrollView style={styles.content}>
    <Text style={styles.subtitle}>{t.warmup.subtitle}</Text>

    {/* Chips muscles ciblés */}
    {muscles.length > 0 && (
      <View style={styles.musclesRow}>
        {muscles.map(m => (
          <View key={m} style={styles.muscleChip}>
            <Text style={styles.muscleChipText}>{m}</Text>
          </View>
        ))}
      </View>
    )}

    {/* Checklist */}
    {suggestions.map((item, index) => (
      <TouchableOpacity
        key={index}
        style={styles.checkItem}
        onPress={() => toggleItem(index)}
      >
        <Ionicons
          name={checked[index] ? 'checkbox' : 'square-outline'}
          size={22}
          color={checked[index] ? colors.primary : colors.textSecondary}
        />
        <Text style={[
          styles.checkText,
          checked[index] && styles.checkTextDone
        ]}>
          {item}
        </Text>
      </TouchableOpacity>
    ))}

    {/* Indicateur progression */}
    <Text style={styles.progress}>
      {checkedCount}/{suggestions.length} {t.warmup.done}
    </Text>
  </ScrollView>
</BottomSheet>
```

**État local dans WarmupChecklistSheet :**
```ts
const suggestions = useMemo(() => generateSuggestions(muscles), [muscles])
const [checked, setChecked] = useState<boolean[]>([])

// Init/reset checked quand suggestions changent
useEffect(() => {
  setChecked(new Array(suggestions.length).fill(false))
}, [suggestions.length])

const toggleItem = useCallback((index: number) => {
  haptics.onSelect()
  setChecked(prev => prev.map((v, i) => i === index ? !v : v))
}, [haptics])

const checkedCount = checked.filter(Boolean).length
```
