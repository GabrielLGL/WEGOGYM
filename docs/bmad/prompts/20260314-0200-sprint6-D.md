# Rapport — Records de Durée (#88) — Groupe D — 20260314-0200

## Objectif
Ajouter une section "Records" en haut de `StatsDurationScreen` affichant la séance la plus courte, la plus longue, et la durée moyenne de l'utilisateur. Ajouter aussi un badge "Record personnel" sur la séance la plus longue dans la liste historique existante.

## Fichiers concernés
- `mobile/src/screens/StatsDurationScreen.tsx` — MODIFIER : section records + badge liste
- `mobile/src/i18n/fr.ts` — MODIFIER : ajouter clés `statsDuration.records.*`
- `mobile/src/i18n/en.ts` — MODIFIER : même structure

## Contexte technique

### Stack & Architecture
- React Native + Expo 52 + TypeScript strict (Fabric / New Architecture)
- WatermelonDB (offline-first) — données déjà disponibles dans `StatsDurationScreenBase`
- `useColors()` → `ThemeColors` depuis `mobile/src/contexts/ThemeContext.tsx`
- `useLanguage()` → `{ t }` depuis `mobile/src/contexts/LanguageContext.tsx`
- **Ne pas modifier les `withObservables` ni les props existantes**

### Structure existante dans StatsDurationScreenBase
```ts
// Props reçues (déjà disponibles)
histories: History[]

// Déjà calculé
const stats = useMemo(() => computeDurationStats(histories), [histories])

// DurationStats (depuis mobile/src/model/utils/statsTypes.ts)
// stats.avgMin: number        — durée moyenne en minutes
// stats.minMin: number        — séance la plus courte
// stats.maxMin: number        — séance la plus longue

//   (toutes les séances avec durée, triées par date décroissante)
```

**`formatDuration(minutes: number)`** est déjà importé depuis `statsDuration.ts` dans ce fichier.

```tsx
{/* ── Records de durée ── */}
```

### Badge "Record" dans la liste historique

Dans le rendu de la liste `stats.historyAll` (autour de la ligne 240-280), identifier la séance la plus longue et lui ajouter un badge.

**Dans le useMemo ou juste avant le render, calculer :**
```ts
const longestHistoryId = useMemo(
  () => stats.historyAll.length > 0
    ? stats.historyAll.reduce((a, b) => a.durationMin > b.durationMin ? a : b).id
    : null,
  [stats.historyAll],
)
```

**Dans le rendu de chaque item de la liste, ajouter conditionnellement le badge :**
```tsx
{entry.id === longestHistoryId && (
  <View style={styles.longestBadge}>
    <Text style={styles.longestBadgeText}>{t.statsDuration.records.longestBadge}</Text>
  </View>
)}
```

**Styles supplémentaires :**

```ts
longestBadge: {

},
longestBadgeText: {
  fontSize: fontSize.caption,
  color: '#F59E0B',
  fontWeight: '600',
},
```
