## Consignes pour chaque ecran

### Regles generales
1. **Lire le fichier source** avant d'ecrire le test pour comprendre les props attendues
2. Importer la version `*Base` (pas le default export wrapped)
3. Tester au minimum :
   - Rendu sans crash avec des props vides/minimales
   - Affichage de l'empty state quand il n'y a pas de donnees
   - Affichage correct avec des donnees de test
4. Mock `expo-haptics` et `../../model/index` systematiquement
5. Mock `react-native-svg` si le screen l'utilise : `jest.mock('react-native-svg', () => ({ Svg: 'Svg', Circle: 'Circle', Line: 'Line', Rect: 'Rect', Text: 'Text', G: 'G' }))`
6. Utiliser `as never` pour caster les props mock si le type est complexe
