## Implémentation

### durationPredictorHelpers.ts

```ts
export interface DurationPrediction {
  estimatedMinutes: number      // durée estimée en minutes
  confidence: 'high' | 'medium' | 'low'
  basedOnSessions: number       // nombre de séances utilisées pour la prédiction
  breakdown: {
    exerciseTime: number        // minutes estimées d'exercice
    restTime: number            // minutes estimées de repos
  }
  range: {
    min: number                 // borne basse (minutes)
    max: number                 // borne haute (minutes)
  }
}

/**
 * Prédit la durée d'une séance basé sur l'historique.
 * Si pas d'historique pour cette session → estimation par défaut basée sur le nombre d'exercices.
 */
export function predictSessionDuration(
  sessionExercises: Array<{ exerciseId: string; sets: number; reps: number }>,
  histories: Array<{ createdAt: Date | number; completedAt: Date | number | null; sessionId: string; deletedAt: Date | null; isAbandoned: boolean }>,
  sessionId: string,
): DurationPrediction
```

**Algorithme :**
1. Trouver les histories passées de cette même session (même sessionId)
   - Filtrer : non supprimées, non abandonnées, avec `completedAt` non null
2. **Si ≥ 3 historiques trouvés** (confiance haute) :
   - Calculer la durée de chaque (completedAt - createdAt, en minutes)
   - `estimatedMinutes` = moyenne des durées
   - `range.min` = min des durées
   - `range.max` = max des durées
   - `confidence` = 'high'
3. **Si 1-2 historiques** (confiance moyenne) :
   - Même calcul, `confidence` = 'medium'
4. **Si 0 historiques** (confiance basse) :
   - Estimation par défaut :
     - `exerciseTime` = nombre d'exercices × sets moyens × 0.75 min/set
     - `restTime` = nombre d'exercices × sets moyens × 1.5 min/rest
     - `estimatedMinutes` = exerciseTime + restTime
   - `range` = ±25%
   - `confidence` = 'low'
5. `breakdown.exerciseTime` = estimatedMinutes × 0.35
6. `breakdown.restTime` = estimatedMinutes × 0.65

### Section UI dans SessionDetailScreen

Ajouter **avant** le bouton "Commencer l'entraînement". Consulter le fichier pour l'emplacement exact — chercher le bouton qui lance le workout.

```tsx
{prediction && (
  <View style={styles.durationPrediction}>
    <View style={styles.durationRow}>
      <Ionicons name="time-outline" size={18} color={colors.textSecondary} />
      <Text style={styles.durationEstimate}>
        {t.sessionDetail.duration.estimate}: ~{prediction.estimatedMinutes} min
      </Text>
    </View>
    <View style={styles.durationDetails}>
      <Text style={styles.durationRange}>
        {prediction.range.min}-{prediction.range.max} min
      </Text>
      <Text style={styles.durationConfidence}>
        {t.sessionDetail.duration.confidence[prediction.confidence]}
      </Text>
    </View>
    {prediction.basedOnSessions > 0 && (
      <Text style={styles.durationBasis}>
        {t.sessionDetail.duration.basedOn.replace('{n}', String(prediction.basedOnSessions))}
      </Text>
    )}
  </View>
)}
```
