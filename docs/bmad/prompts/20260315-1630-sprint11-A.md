## Implémentation

### muscleRecoveryHelpers.ts

```ts
export type RecoveryStatus = 'fresh' | 'recovered' | 'recovering' | 'fatigued'

export interface MuscleRecoveryEntry {
  muscle: string
  status: RecoveryStatus
  hoursSinceLastTrained: number | null  // null si jamais entraîné
  recoveryPercent: number               // 0-100 (100 = frais)
  lastVolume: number                    // volume de la dernière séance pour ce muscle
}

/**
 * Estime la récupération par groupe musculaire.
 * Basé sur le temps depuis le dernier entraînement + volume.
 * Temps de récupération estimé : 48h (petits muscles) à 72h (grands muscles).
 */
export function computeMuscleRecovery(
  sets: Array<{ weight: number; reps: number; exerciseId: string; createdAt: Date | number }>,
  exercises: Array<{ id: string; muscles: string[] }>,
): MuscleRecoveryEntry[]
```

**Algorithme :**
1. Temps de récupération de base par muscle :
   - Grands muscles (Pecs, Dos, Quadriceps, Ischios) : 72h
   - Moyens muscles (Epaules, Trapèzes) : 60h
   - Petits muscles (Biceps, Triceps, Mollets, Abdos) : 48h
   - Cardio : 24h
2. Pour chaque muscle de MUSCLES_LIST :
   - Trouver le dernier set (le plus récent) qui implique ce muscle
   - Calculer `hoursSinceLastTrained`
   - Calculer le volume de la dernière session (même jour) pour ce muscle
   - Ajuster le temps de récupération : si volume > 2000kg pour ce muscle, ajouter 12h
3. `recoveryPercent` = min(100, (hoursSinceLastTrained / tempsRecuperation) × 100)
4. **Status** :
   - recoveryPercent ≥ 100 → 'fresh'
   - recoveryPercent ≥ 70 → 'recovered'
   - recoveryPercent ≥ 40 → 'recovering'
   - < 40 → 'fatigued'
5. Trier : fatigued en premier, fresh en dernier

**Fonction couleur :**
```ts
function getRecoveryColor(status: RecoveryStatus, colors: any): string {
  switch (status) {
    case 'fresh': return colors.primary
    case 'recovered': return '#10B981'   // vert
    case 'recovering': return '#F59E0B'  // jaune
    case 'fatigued': return colors.danger
  }
}
```
