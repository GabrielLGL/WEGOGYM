# Rapport — Audit 7 Site Kore — Groupe A : Footer accessibilité — 20260227-2100

## Objectif
2 corrections d'accessibilité dans `FooterSection.tsx` :
1. Envelopper les liens du footer dans une balise `<nav aria-label="Navigation pied de page">` pour que les lecteurs d'écran identifient cette zone de navigation
2. Ajouter un `<h2 className="sr-only">Kore</h2>` avant le logo pour que les lecteurs d'écran aient un heading visible pour la section footer

## Fichiers concernés
- `web/src/components/sections/FooterSection.tsx`

## Contexte technique
- Composant Server Component Next.js — pas de `"use client"`
- Le `<footer>` HTML5 a déjà un role implicite `contentinfo` — c'est correct
- Les 4 liens (Fonctionnalités, Tarifs, Contact, Confidentialité) sont actuellement dans un `<div>` sans sémantique de navigation
- `npx tsc --noEmit` (depuis `web/`) doit rester à 0 erreur

## Étapes

### 1. `<nav>` sur les liens de navigation (lignes 7–20)

Remplacer le `<div className="flex justify-center gap-6 mb-6 flex-wrap">` qui entoure les liens par une `<nav>` :

```tsx
// AVANT
<div className="flex justify-center gap-6 mb-6 flex-wrap">
  <a href="#features" ...>Fonctionnalités</a>
  <a href="#pricing" ...>Tarifs</a>
  <a href="mailto:contact@kore-app.com" ...>Contact</a>
  <a href="/privacy" ...>Confidentialité</a>
</div>

// APRÈS
<nav aria-label="Navigation pied de page" className="flex justify-center gap-6 mb-6 flex-wrap">
  <a href="#features" ...>Fonctionnalités</a>
  <a href="#pricing" ...>Tarifs</a>
  <a href="mailto:contact@kore-app.com" ...>Contact</a>
  <a href="/privacy" ...>Confidentialité</a>
</nav>
```

Uniquement changer la balise `div` → `nav` et ajouter `aria-label="Navigation pied de page"`. Les classes CSS restent identiques.

### 2. Heading sr-only avant le logo (ligne 54)

Insérer un `<h2 className="sr-only">Kore</h2>` immédiatement avant la `<div>` du logo :

```tsx
// AVANT
<div className="flex items-center justify-center gap-2 mb-6">
  <KoreLogo size={24} gradientId="footerGrad" className="!animate-none" />
  <span className="gradient-text font-black text-sm tracking-widest">KORE</span>
</div>

// APRÈS
<h2 className="sr-only">Kore</h2>
<div className="flex items-center justify-center gap-2 mb-6">
  <KoreLogo size={24} gradientId="footerGrad" className="!animate-none" />
  <span className="gradient-text font-black text-sm tracking-widest">KORE</span>
</div>
```

## Contraintes
- Ne modifier QUE ce qui est indiqué (balise div→nav + ajout h2)
- Ne pas modifier les classes Tailwind existantes
- Ne pas modifier les liens, icônes sociales, ou le copyright

## Critères de validation
- `npx tsc --noEmit` (depuis `web/`) → 0 erreur
- Lecteur d'écran : footer annoncé comme "Navigation pied de page" pour la zone liens
- Aucune régression visuelle (nav = inline, même rendu que div)
