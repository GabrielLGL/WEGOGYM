## Implémentation

### motivationHelpers.ts

```ts
export type MotivationContext =
  | 'returning_after_long'    // > 2× la fréquence habituelle
  | 'slight_drop'             // 1.5–2× la fréquence habituelle
  | 'keep_going'              // dans les normes mais streak en jeu
  | null                      // aucun message à montrer

export interface MotivationData {
  context: MotivationContext
  daysSinceLastWorkout: number
  avgDaysBetweenWorkouts: number
}

/**
 * Détecte le contexte de motivation depuis l'historique.
 * Retourne null si moins de 3 séances (pas assez de données).
 */
export function computeMotivation(histories: History[]): MotivationData | null
```

**Algorithme :**
1. Si `histories.length < 3` → retourner `null`
2. Trier par `startTime` décroissant
3. `lastWorkoutDate` = `histories[0].startTime.getTime()`
4. `daysSinceLastWorkout` = `Math.floor((Date.now() - lastWorkoutDate) / DAY_MS)`
5. Calculer `avgDaysBetweenWorkouts` sur les 10 dernières séances :
   - Prendre les dates des séances triées
   - Calculer les intervalles consécutifs
   - Moyenne des intervalles
6. Logique :
   - `daysSinceLastWorkout < avgDaysBetweenWorkouts` → `null` (tout va bien)
   - `daysSinceLastWorkout >= avgDaysBetweenWorkouts * 2` → `'returning_after_long'`
   - `daysSinceLastWorkout >= avgDaysBetweenWorkouts * 1.5` → `'slight_drop'`
   - `daysSinceLastWorkout >= avgDaysBetweenWorkouts - 0.5` ET streak actif → `'keep_going'`
   - Sinon → `null`

**Import à ajouter :**
```ts
import { DAY_MS } from '../model/constants'
```

### Intégration dans HomeScreen

**Import à ajouter :**
```ts
import { computeMotivation } from '../model/utils/motivationHelpers'
```

**useMemo dans HomeScreenBase** (après les flashbacks) :
```ts
const motivationData = useMemo(
  () => computeMotivation(histories),
  [histories],
)
```

**Placement dans le JSX :** Ajouter **avant** `{/* ── Weekly Report Card ── */}` :
```tsx
{/* ── Motivation contextuelle ── */}
{motivationData && (
  <View style={styles.motivationCard}>
    <View style={styles.motivationHeader}>
      <Ionicons
        name={motivationData.context === 'returning_after_long' ? 'heart-outline' : 'flame-outline'}
        size={20}
        color={colors.primary}
      />
      <Text style={styles.motivationTitle}>
        {t.motivation[motivationData.context + 'Title' as keyof typeof t.motivation]}
      </Text>
    </View>
    <Text style={styles.motivationText}>
      {t.motivation[motivationData.context + 'Message' as keyof typeof t.motivation]
        ?.replace('{n}', String(motivationData.daysSinceLastWorkout))}
    </Text>
  </View>
)}
```

**Note :** Utiliser un helper de lookup plutôt que les clés dynamiques si TypeScript se plaint. Alternative propre :
```tsx
const motivationTitle = motivationData
  ? { returning_after_long: t.motivation.returningAfterLongTitle,
      slight_drop: t.motivation.slightDropTitle,
      keep_going: t.motivation.keepGoingTitle }[motivationData.context!]
  : null

const motivationMessage = motivationData
  ? { returning_after_long: t.motivation.returningAfterLongMessage,
      slight_drop: t.motivation.slightDropMessage,
      keep_going: t.motivation.keepGoingMessage }[motivationData.context!]
      ?.replace('{n}', String(motivationData.daysSinceLastWorkout))
  : null
```
