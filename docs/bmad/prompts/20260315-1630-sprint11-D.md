# Rapport — Prévision de Volume (#104) — Groupe D — 20260315-1630

## Implémentation

### volumeForecastHelpers.ts

```ts
export interface WeeklyVolume {
  weekStart: number        // timestamp lundi
  volume: number           // kg total
  sessions: number
}

export interface VolumeForecast {
  predictedVolume: number  // kg prédit pour la semaine à venir
  confidenceRange: {
    low: number            // borne basse
    high: number           // borne haute
  }
  trend: 'increasing' | 'decreasing' | 'stable'
  trendPercent: number     // % de progression hebdo
  weeklyHistory: WeeklyVolume[]  // 8 dernières semaines
  currentWeekVolume: number      // volume de la semaine en cours
  currentWeekProjection: number  // projection fin de semaine (basé sur jours restants)
  monthlyPace: {
    current: number        // volume ce mois-ci
    projected: number      // projection fin de mois
    averageMonthly: number // moyenne mensuelle historique
  }
}

/**
 * Prédit le volume d'entraînement pour la semaine à venir.
 * Utilise une moyenne pondérée des 8 dernières semaines (semaines récentes = plus de poids).
 */
export function computeVolumeForecast(
  sets: Array<{ weight: number; reps: number; createdAt: Date | number }>,
): VolumeForecast
```

**Algorithme :**
1. Grouper les sets par semaine (lundi → dimanche) sur les 8 dernières semaines
2. Calculer le volume hebdomadaire = Σ(weight × reps)
3. **Moyenne pondérée** : semaine -1 = poids 4, -2 = poids 3, -3 = poids 2, -4 à -8 = poids 1
   - `predictedVolume` = Σ(volume × poids) / Σ(poids)
4. **Intervalle de confiance** : écart-type des 8 semaines
   - `low` = predictedVolume - stdDev
   - `high` = predictedVolume + stdDev
5. **Trend** : régression linéaire sur les volumes hebdomadaires
   - slope > 5% par semaine → 'increasing'
   - slope < -5% → 'decreasing'
   - sinon → 'stable'
6. **Semaine en cours** : volume déjà fait + projection (volume restant = avg journalier × jours restants)
7. **Pace mensuel** : extrapoler le volume du mois basé sur le rythme actuel

const enhance = withObservables([], () => ({
  sets: database.get<WorkoutSet>('sets').query(
    Q.on('histories', Q.and(
      Q.where('deleted_at', null),
      Q.or(Q.where('is_abandoned', null), Q.where('is_abandoned', false)),
    )),
  ).observe(),
}))
